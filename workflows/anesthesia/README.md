## 🏷️ 步驟代碼系統

### 代碼格式

```
[Phase 字母]-[序號]      → 基線步驟      例：A-01, C-05, G-17
[Phase 字母]-[類型][序號] → 類型專屬步驟  例：A-D01 (日間專屬), B-E01 (E刀專屬)
```

- **Phase 字母**：A ~ L（對應 12 個流程階段）
- **序號**：兩位數字，從 01 開始
- **類型前綴**：`E` = Emergency, `U` = Urgent, `D` = Day Surgery

### 狀態標記

| 標記 | 意義 | 說明 |
|:----:|------|------|
| ✅ | 同基線 | 步驟內容與常規刀完全相同 |
| ⚡ | 改動 | 步驟存在但內容有修改 |
| ⏭️ | 跳過 | 該類型不執行此步驟 |
| 🆕 | 新增 | 該類型專屬步驟，基線無此步驟 |
