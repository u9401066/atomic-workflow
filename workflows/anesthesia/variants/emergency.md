# 🔴 E刀（Emergency）流程

> **定義**：生命威脅或器官存亡，必須 **立即** 進入手術室  
> **基線參照**：[常規刀基線](../anesthesia-workflow-granular.md) | [步驟代碼索引](README.md)  
> **核心差異**：無術前門診 → 床邊極簡評估 → ED/病房直送 OR → 全胃 RSI → 可能→ ICU  
> **標記說明**：✅同基線 | ⚡改動 | ⏭️跳過 | 🆕新增

---

## Phase A：急診床邊麻醉評估（取代術前門診）

> ⏱️ 壓縮至 **10–30 分鐘**。無門診、無預約，麻醉醫師直奔急診或病房床邊。

### [A-01] 接到麻醉照會 ⚡改

- **接案方式**：
  - 急診醫師 / 外科值班醫師 **電話直撥** 麻醉科值班
  - 系統同步開立「🔴 緊急麻醉照會」
  - ⚠️ 電話內容必須包含：病人位置、診斷、預計手術、預估時間
- **麻醉醫師反應**：
  - 立即前往（非遠端評估）
  - 通知 OR 值班：「E 刀即將上」→ 刀房開始備刀
  - 通知麻醉護理師 / 助手備藥備機
- **與基線差異**：無門診排程、無提前照會，所有動作同步啟動

### [A-02] 調閱電子病歷 ⚡改

- **快速瀏覽重點**（邊走邊看手機/平板）：
  - 過敏史（最優先）
  - 目前用藥（抗凝血劑、胰島素、心臟藥物）
  - 重大共病（心臟病/COPD/腎衰竭/肝硬化）
  - 前次麻醉紀錄（困難氣道？MH？）
- **可跳過**：完整門診紀錄回顧、既往會診紀錄
- ⚠️ 如系統查不到（外院轉入/初次就醫）→ 全靠床邊問診 + 家屬提供

### [A-03] 查閱檢驗報告 ⚡改

- **必看**（已有的立刻看）：
  - CBC（Hb！→ 備血決策）
  - BMP / 電解質（K⁺！→ Succinylcholine 禁忌判斷）
  - Coagulation（PT/INR → 區域麻醉可行性）
  - Type & Screen / Crossmatch（已送？結果？）
  - Blood gas（如有 → 酸鹼 + lactate）
- **缺的怎麼辦**：
  - 基本檢驗未做 → **立刻抽血送檢**（邊送邊準備）
  - 結果還沒出 → 記錄「pending」→ 一出來立刻看
  - ECG → 有就看，沒有就靠 monitor lead II 觀察
  - CXR → 有就看，沒有不等
- **與基線差異**：不等報告齊全才行動，邊做邊看

### [A-04] 床邊評估 ⚡改

- 確認病人身份（手環 + 口頭，如病人意識不清 → 手環 + 家屬確認）
- 自我介紹：「我是麻醉科醫師，現在要評估您的狀況」
- ⚠️ 病人可能：昏迷、插管中、極度疼痛、躁動、醉酒 → 調整溝通策略

### [A-05] 病史詢問 ⚡改

- **資料來源優先順序**：
  1. 病歷系統（最快）
  2. 床邊病人（如清醒可溝通）
  3. 家屬（電話或在場）
  4. 急診醫師/護理師（轉述）
  5. 轉診單（外院轉來）
- **必問項目**（快問快答版）：
  - 過敏？（藥物/食物/latex）
  - 前次手術/麻醉有沒有問題？
  - 心臟病？呼吸問題？
  - 糖尿病？胰島素？
  - 吃抗凝血劑？哪一種？最後一次吃？
  - 最後一餐什麼時候？吃了什麼？（NPO assessment）
  - 抽菸？喝酒？嚼檳榔？
  - 鬆動牙齒？假牙？
- **可省略**：完整系統性回顧、家族 MH 詳細追問（但如有異常仍需注意）

### [A-06] 氣道評估 ⚡改

- **快速床邊評估**（30 秒內）：
  - Mallampati（病人能張嘴配合就看）
  - 張口度（目測 > 3 指？）
  - 頸椎活動（⚠️ 頸椎損傷疑慮 → 不做！維持固定）
  - 牙齒（鬆動/假牙 → 快速記錄）
  - 鬍鬚（面罩密合問題）
  - 下顎前突能力（如配合度允許）
- ⚠️ **無法評估的情境**：
  - 昏迷/插管中 → 假設困難氣道 → 備齊設備
  - 顏面外傷 → 假設困難氣道
  - 頸椎固定 → 限制頸椎活動 → video laryngoscope 優先

### [A-07] 心肺檢查 ⚡改

- 心音聽診（快速：murmur? irregular?）
- 肺音聽診（wheeze? crackles? 對稱?）
- ⚠️ 急診環境嘈雜 → 聽診品質可能不佳 → 輔以監測數據
- 周邊水腫 / JVP → 快速目視
- 脊椎（如考慮 RA）→ 通常 E 刀不優先做 RA

### [A-08] 風險評估 ⚡改

- ASA 分級（快速判斷）
  - E 刀自動加上 **E 後綴**（如 ASA 3E）
- 心臟風險 → 快速判斷（不做完整 RCRI scoring）
- ⚠️ **全胃風險評估 = 預設 YES**：
  - 除非有明確證據已禁食 ≥ 6 hr（固體）/ ≥ 2 hr（清水）
  - 外傷/急腹症/腸阻塞/孕婦 → **一律全胃處理**
  - → RSI 準備
- 困難氣道風險 → 判定 Plan A/B/C
- 大出血風險 → 通知血庫準備

### [A-09] 制定麻醉計畫 ⚡改

- **優先考量**：
  - 全胃 → **RSI（Rapid Sequence Induction）** 
  - 血行動力學不穩 → Ketamine / Etomidate 取代 Propofol
  - 大出血進行中 → 大口徑 IV × 2 + A-line + 可能 CVC
  - 頸椎受傷 → Awake fibreoptic intubation 或 Video laryngoscope with in-line stabilization
- **麻醉方式**：
  - 大部分 E 刀 → **GA（全身麻醉）**
  - 少數可 RA 的情境（如下肢骨折固定 + 生命徵象穩定）→ 需個案判斷
- **計畫簡化版記錄**：可用口頭+心中計畫，術後再補完整紀錄

### [A-10] 向病人說明 ⚡改

- **病人清醒能溝通**：
  - 快速版說明（1–2 分鐘內）
  - 「我們需要幫您全身麻醉進行手術，會有哪些風險...」
  - 回答關鍵問題
- **病人意識不清 / 無法溝通**：
  - 向家屬說明（在場 or 電話）
  - 記錄溝通對象 + 時間
- ⚠️ 時間壓力下仍需完成知情同意的法律義務

### [A-11] 簽署麻醉同意書 ⚡改

- **情境處理**：
  - 🅰️ 病人清醒 + 有行為能力 → 正常簽署
  - 🅱️ 病人無法簽 + 家屬在場 → 家屬簽署（法定代理人 / 配偶 / 直系親屬）
  - 🅲️ 病人無法簽 + 家屬電話同意 → 記錄通話時間 + 對象 + 見證人簽名
  - 🅳️ **無家屬 + 病人無法簽** → 依法緊急醫療處置：
    - 兩位專科醫師共同簽署「緊急醫療處置同意書」
    - 記錄無法取得同意的原因
    - 術後補簽
- ⚠️ 絕不因同意書問題延誤搶救

### [A-12] 簽署自費同意書 ⚡改

- 緊急情況下自費品項使用 → 事後補簽
- 優先使用 → 事後說明費用
- 記錄緊急使用的原因

### [A-13] 填寫麻醉術前評估表 ⚡改

- **術前**：記錄已完成的評估項目（可能不完整）
- **術後補齊**：手術結束後回頭補完整表格
- ⚠️ 至少在進 OR 前記錄：ASA 分級 + 氣道評估 + 過敏 + 麻醉計畫

### [A-14] 安排額外檢查或會診 ⏭️跳過

- E 刀不等額外檢查
- 如術中發現需要 → 術後安排
- 唯一例外：術前 Echo（如高度懷疑 cardiac tamponade → 可能影響麻醉策略）→ 床邊快速 Echo（FATE/FOCUS）

## Phase B：急診端準備（取代病房端準備）

> E 刀無病房準備流程。所有準備在 **ED 同步進行** 或 **在 OR 內完成**。

### [B-01] ~ [B-04] 病房護理站流程 ⏭️跳過

> 無病房護理站參與。由 ED 護理師 + 麻醉團隊同步處理。

### [B-E01] ED 同步準備 🆕

- **ED 護理師執行**（與麻醉評估同步）：
  - 確認 IV 通暢（已建 → 確認 size 足夠；未建 → 建立，至少 18G）
  - 第二條 IV line（如預期大出血 → 18G 或更粗 × 2）
  - 備血單送出（Type & Screen + Crossmatch）
  - 抗生素依外科醫囑給予
  - 手環確認在手上
  - 手術部位標記（如外科已標 → 確認；未標 → 要求外科標記）
    - ⚠️ 急腹症 / 正中開刀 → 可能無需標記側別
- **病人更衣**：ED 病人通常已穿病人服
  - 快速移除飾品/假牙等（如時間允許）
  - ⚠️ 外傷病人 → 衣物已剪開 / 頸圈 on → 保持
- **文件**：
  - 急診病歷 + 影像 + 檢驗 → 一起帶到 OR
  - 同意書（已簽或待補）→ 隨行
  - ⚠️ 文件不齊全 → **不影響出發**，邊走邊處理

### [B-05] ~ [B-07] 病人準備 ⚡改

- 簡化版：在 ED stretcher 上完成（不換推床）
- 假牙快速取下（時間允許的話）
- ⚠️ 指甲油/光療 → 來不及移除就記錄「未移除」→ 改用前額/耳垂 SpO2
- 禁食確認 → **直接問「最後一餐？」**，不足 → RSI 計畫不變

### [B-08] IV 確認 ⚡改

- ED 通常已建立 → 確認位置 + 口徑 + 通暢
- 口徑不足（< 18G 且預期大出血）→ 到 OR 後追加
- ⚠️ ED IV 可能在不理想位置（足背/手背）→ 到 OR 後評估是否重建

### [B-09] 辨識手環 ✅同基線

### [B-10] 備血確認 ⚡改

- **已完成 crossmatch** → 確認血品備妥，通知血庫「E 刀即將上」
- **未完成 crossmatch** → 立刻送 → 告知血庫優先處理
- **大量出血中** → 啟動 **Massive Transfusion Protocol (MTP)**
  - 不等 crossmatch → 先用 O-negative PRBC
  - 通知血庫持續備血
- ⚠️ 血庫電話號碼 + MTP 啟動關鍵詞 → 麻醉醫師必須熟悉

### [B-11] ~ [B-14] 文件+出發 ⚡改

- **文件整理簡化**：急診護理師把現有文件裝袋 → 跟人走
- **通知 OR**：麻醉醫師直接通知（或護理站代通知）「E 刀 XX 分鐘內到」
- **不等傳送人員**：ED 護理師+麻醉團隊直接推送
- **搬運前確認簡化**：
  - IV 通暢 ✓
  - 監測器材隨行（portable monitor + O2）
  - 急救藥物帶著走（Epi 安瓶 + Atropine）
  - ⚠️ 插管中的病人 → Ambu bag 通氣 + SpO2 監測一路到 OR

## Phase C：直送開刀房（取代標準報到）

> E 刀通常 **跳過前台報到**，由 ED 直送 OR 門口。

### [C-01] 從 ED 出發 ⚡改

- ED stretcher 直接推走
- 不換推床（除非到 OR 後需換手術台）
- 確認氧氣瓶 / portable monitor / Ambu bag 隨行
- 確認引流管/導尿管固定

### [C-02] 移動路線 ⚡改

- 路線可能不同於病房動線（ED → 手術區的最短路線）
- ⚠️ 外傷 / 失血性休克病人 → 全程跑步推送
- ⚠️ ED 直通 OR 的快速通道（如有）→ 走快速通道

### [C-E01] 通報 OR 前台 🆕

- **電話先行通報**：「E 刀直送，病人 XX，XX 手術，到 XX 號刀房」
- 前台直接在系統登錄「已報到」→ 不需停車報到
- 刀房號確認（ED 出發前已由 OR 調度分配）

### [C-03] ~ [C-06] 前台報到 ⚡改

- **大部分跳過** — 不停前台
- 前台人員收到電話通報後直接登錄
- 生理參數核對 → **在 OR 內完成**（接上 monitor 後第一組數據即為報到數據）
- 文件核對 → **在 OR 內由麻醉護理師完成**（或術後補）
  - ⚠️ WHO Sign-In 仍需執行（在 OR 內，不可跳過）
- NPO 確認 → 已在 ED 確認 → 假設 RSI

### [C-07] 系統登錄 ⚡改

- 由前台人員遠端操作（依通報電話內容）
- 或由 OR 護理師在刀房內系統登錄

### [C-08] 判斷去向 ⚡改

- **一律直進指定刀房** — 不經等候區
- ⚠️ 指定刀房尚未清潔完成 → 等候在 OR 走廊 → 麻醉醫師全程陪同監測

## Phase D：直接進入手術室（跳過等候區）

### [D-01] ~ [D-07] 等候區流程 ⏭️跳過

> E 刀不進等候區。直接進 OR。

### [D-08] 推至刀房門口 ⚡改

- 核對刀房號 → 推入
- 可能在走廊短暫等待（前一台清潔中）→ 全程監測不中斷

### [D-09] 刀房門口確認 ⚡改

- 快速三方核對（可在推入後、WHO Sign-In 前完成）
- 不因確認延誤推入

## Phase E：手術室內 — 病人安頓 & 監測建立

> 與基線大致相同，但節奏 **加速**，部分步驟同步進行。

### [E-01] ~ [E-04] 推床進入 + 移位 + 固定 ✅同基線

### [E-05] 確認病人舒適 ⚡改

- 昏迷/插管/極度疼痛 → 無法詢問 → 直接進行
- 清醒病人 → 簡短安撫

### [E-06] ~ [E-08] ECG + BP + SpO2 ✅同基線

> ⚠️ 加速貼上！多人同步操作（一人貼 ECG，一人上 BP cuff，一人夾 SpO2）

### [E-09] 拆封自費品項 ⚡改

- 緊急情況下 → 先用再說 → 自費同意術後補簽
- BIS 不一定使用（如 hemodynamically unstable → 全心力在穩定 VS）

### [E-10] ~ [E-13] BIS + 保溫毯 + 溫度 + NMT ✅同基線

### [E-14] A-line ⚡改

- E 刀更常需要 A-line（血行動力學不穩 / 大出血預期）
- 可能在 **誘導前** 清醒狀態下先建立（如時間允許 + 血壓不穩需 beat-to-beat 監測）
- 或 **誘導後** 建立（如需快速進行）

### [E-15] CVC ⚡改

- 更常需要（升壓藥持續輸注 / 多管路需求）
- 通常 **誘導後** 建立
- ⚠️ 考慮 **大口徑引導器鞘（Cordis / MAC）** 取代標準 CVC（快速輸液需求）

---

## Phase F：麻醉準備 ⚡改

> 與基線相同步驟，但 **時間壓力**。OR 護理師/技術員在病人到達前已開始備刀。

### [F-01] ~ [F-06] 機器檢查 ✅同基線

> ⚠️ E 刀房的麻醉機應 **每日首台前** 已完成自檢。第二台以上 → 確認方式簡化。

### [F-07] 誘導藥物 ⚡改

- **RSI 藥物組合**（E 刀預設）：
  - Ketamine 1–2 mg/kg（或 Etomidate 0.2–0.3 mg/kg）→ 取代 Propofol（血壓不穩時）
  - Propofol → 僅血壓穩定時使用
  - Rocuronium **1.2 mg/kg**（RSI 劑量，非 0.6）
  - 或 Succinylcholine 1–1.5 mg/kg（無禁忌時）
  - Fentanyl → 可能減量或不給（低血壓風險）
- ⚠️ **藥物標籤仍需貼**（不因緊急而省略安全步驟）

### [F-08] 急救藥物 ✅同基線

### [F-09] 持續輸注 ⚡改

- Vasopressor infusion 更常預備（Norepinephrine / Vasopressin）
- TXA 提前備好

### [F-10] 止吐藥 ⚡改

- 可能延後或省略（非優先）

### [F-11] 特殊藥物 ⚡改

- MTP 藥物備齊（TXA / Calcium / 溫熱輸液）
- Intralipid 確認位置（如果用了大量局麻藥）

### [F-12] 輸液 ⚡改

- **多條輸液線**：至少兩條 large-bore IV
- **加壓輸液系統**（Level 1 / Belmont）提前架設
- 溫熱晶體液 ready

### [F-13] ~ [F-17] 氣道設備 ✅同基線

> ⚠️ **Video laryngoscope 預設準備**（E 刀困難氣道發生率高於常規）  
> ⚠️ **抽吸裝置務必確認**（RSI = 嘔吐風險 → 抽吸必須即時可用）

## Phase G：麻醉誘導

### [G-01] ~ [G-06] WHO Sign-In ⚡改

- **仍需執行**，但可加速
- 加入額外確認：「Full stomach → RSI planned ✓」
- 「MTP on standby ✓」（如適用）

### [G-07] ~ [G-08] 面罩 + O2 ✅同基線

### [G-09] ~ [G-10] 預氧合 ⚡改

- 時間可能不足 3 分鐘 → 用 **8 次深呼吸法**（vital capacity breaths）
- ⚠️ 配合 **HFNC 15 L/min** 同步（apneic oxygenation during RSI）
- ⚠️ 嘔吐風險 → 抽吸管在右手旁可及

### [G-E01] RSI 快速誘導程序 🆕

> **RSI = 不做中間面罩通氣 → 從誘導到插管連續執行**

1. **Cricoid pressure（Sellick's maneuver）**
   - 助手施壓 cricoid cartilage → 壓住食道 → 減少被動反流
   - ⚠️ 爭議：部分指引已不強力推薦，但多數台灣臨床仍執行
   - ⚠️ 如插管困難 → 放鬆 cricoid → 改善聲門暴露

2. **誘導藥物快速推注**（無等待間隔）
   - Fentanyl → 減量或不給（血壓不穩時）
   - 誘導劑 → Ketamine / Etomidate / Propofol → 快速 push
   - 肌鬆劑 → **立即接著推**（Rocuronium 1.2 mg/kg 或 Succinylcholine 1–1.5 mg/kg）

3. **不做面罩通氣**（= modified RSI 則可輕柔面罩通氣，低壓 < 20 cmH2O）
   - 等待 45–90 秒（肌鬆作用）
   - 持續 cricoid pressure
   - SpO2 密切監測

4. **立即插管**
   - 首選 Video laryngoscope（E 刀困難氣道高風險）
   - 一次成功最重要
   - ⚡ 失敗 → Plan B（Bougie / 換 blade / 不同姿勢）
   - ⚡ 二次失敗 → Plan C（LMA / i-gel → 暫時通氣）
   - ⚡ CICO（Can't Intubate, Can't Oxygenate）→ 外科氣道（Cricothyrotomy）

5. **確認管位**（同基線 G-24，5 步驟）
6. **接呼吸器**

### [G-11] ~ [G-34] 其他誘導/RA/MAC 步驟

- GA 插管：由 [G-E01] RSI 流程取代標準 G-11 ~ G-19
- G-20 以後（插管後）：✅同基線
- RA（G-31 ~ G-33）：E 刀極少使用 RA → 跳過或個案決定
- MAC（G-34）：E 刀極少使用 MAC

### [J-02] 調整麻醉深度 ⚡改

## Phase K：PACU / ICU 轉送

### [K-01] ~ [K-03] 轉送 ⚡改

- **去向決策**：
  - 🅰️ **→ ICU**（大部分嚴重 E 刀）：帶管 + 持續鎮靜 + 升壓藥
  - 🅱️ **→ PACU**（穩定的 E 刀）：同基線
- **ICU 轉送注意**：
  - Transport monitor 持續
  - 升壓藥/鎮靜輸注不中斷
  - Ambu bag 隨行（帶管病人）
  - 完整交班準備

### [K-04] SBAR 交班 ⚡改

- 交班對象：
  - → ICU：交給 ICU 主治醫師 + ICU 護理師
  - → PACU：同基線
- **ICU 交班額外內容**：
  - 術中出血總量 + 輸血量（PRBC/FFP/PLT/Cryo）
  - 最新 ABG + lab
  - 升壓藥種類 + 劑量 + trend
  - 是否預計二次手術 + 時間
  - 帶管狀態（ETT size / 深度 / 鎮靜藥物 / 呼吸器設定建議）

### [K-05] ~ [K-09] ICU 接手後由 ICU 團隊管理 / PACU 同基線

### [K-06] Aldrete Score ⚡改

- ICU → 不用 Aldrete（用 ICU 自己的評估工具）
- PACU → 同基線

### [K-07] 監測 ⚡改

- ICU → 持續 ICU level 監測
- PACU → 同基線但觀察密度可能增加

### [K-10] ~ [K-13] 出室 ⚡改

- ICU → 無出室流程（ICU 自主管理後續）
- PACU → 同基線但出室條件更嚴格

## Phase L：術後照護 & 紀錄結案

### [L-01] APS ⚡改

- ICU 病人 → APS 可能與 ICU 共管
- 疼痛方案可能含持續鎮靜（如帶管 + Propofol / Dexmedetomidine infusion）

### [L-02] 術後訪視 ⚡改

- ICU 病人 → 麻醉團隊至 ICU 訪視
- 評估拔管時機（如帶管轉入）
- 評估是否需二次手術的麻醉計畫

### [L-03] ~ [L-07] ✅同基線

### [L-08] 品質紀錄 ⚡改

- E 刀額外記錄項目：
  - 門診→OR 反應時間（decision to incision time）
  - RSI 使用紀錄
  - MTP 啟動紀錄
  - 非計畫性帶管轉 ICU 紀錄
  - 品質指標追蹤（30-day mortality for emergency cases）
