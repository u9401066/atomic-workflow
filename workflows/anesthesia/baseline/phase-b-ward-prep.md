# Phase B：手術日 — 病房端準備

> **步驟代碼**：B-01 ~ B-14（共 14 步）  
> **主要執行者**：👩‍⚕️ 病房護理師  
> **協助人員**：👨‍⚕️ 主治醫師（手術部位標記） ／ 傳送人員  
> **時機**：手術日早晨 ~ 出發前往開刀房  
> **基線**：🟢 常規刀（Elective）

---

## B1. 術前再確認（護理站）

### [B-01] 護理師確認手術排程
**執行者**：👩‍⚕️ 病房護理師

- 查看當日手術排程表
- 確認手術時間（第幾台）
- 確認該台手術預估開始時間
- 通知病人預計送刀房時間

### [B-02] 確認禁食狀態
**執行者**：👩‍⚕️ 病房護理師

- 詢問病人最後進食時間
- 詢問最後喝水時間
- 記錄於護理紀錄
- ⚠️ 若未達禁食標準 → 通知手術室 + 麻醉科
- 👩‍⚕️ **提醒**：
  - 前一晚已衛教禁食時間，此時為「再確認」
  - 部分病人在深夜偷吃 → 需直接詢問「有沒有偷喝一口水或吃東西？」
  - 記錄格式：最後進食 _____（時間＋內容）；最後飲水 _____（時間）

### [B-03] 確認術前用藥
**執行者**：👩‍⚕️ 病房護理師

- 核對麻醉評估表的用藥指示
- 該吃的藥準備好 + 小杯水（< 30 ml）
- 該停的藥確認已停
- 胰島素劑量調整依指示
- 記錄給藥時間
- 👩‍⚕️ **常見情境**：
  - 降壓藥（BB / CCB）→ 早上照吃，小口水送服
  - Metformin → 手術當天停
  - Warfarin → 確認已停 3–5 天
  - DOAC → 確認已停 24–48 hr
  - 胰島素 → 依醫囑減量或不打

### [B-04] 術前預防性抗生素（如 protocol 在病房給）
**執行者**：👩‍⚕️ 病房護理師

- 確認醫囑
- 確認過敏史（Penicillin / Cephalosporin）
- 給藥（目標：切刀前 60 分鐘內）
- 記錄給藥時間
- ⚠️ 有些醫院 protocol 是到 OR 才給 → 確認院規
- ⚠️ Vancomycin 需 1–2 hr 輸注 → 需更早開始

## B2. 病人準備

### [B-05] 病人更衣
**執行者**：👩‍⚕️ 病房護理師（指導） ／ 病人自行

- 換穿手術衣（開口朝後）
- 內衣褲脫除（依手術部位）
- 提供保暖被單
- 👩‍⚕️ **注意**：
  - 確認手術衣穿法正確（開口朝後，tie 繫好但不過緊）
  - 臥床病人：協助更衣
  - 冬天：額外蓋被保暖

### [B-06] 移除個人物品
**執行者**：👩‍⚕️ 病房護理師（逐一確認）

- 假牙移除 → 放入標示容器交家屬或護理站保管
- 眼鏡 / 隱形眼鏡移除
- 助聽器移除（或保留至 OR，依溝通需要）
- 首飾移除（戒指 / 項鍊 / 耳環 / 手錶）
  - ⚠️ 無法取下的戒指 → 膠帶包覆 + 記錄
- 髮夾 / 髮飾移除
- 活動式假牙 / 牙套移除
- 指甲油 / 光療移除（影響 SpO2 讀數）
- 身上金屬物品移除（如使用電燒）
- 👩‍⚕️ **記錄**：
  - 物品保管清單填寫（品項 + 數量 + 保管方式）
  - 家屬簽收（或護理站保管 + 記錄）
  - ⚠️ 貴重物品爭議預防：清單需病人/家屬確認簽名

### [B-07] 確認手術部位標記
**執行者**：👩‍⚕️ 病房護理師（確認） ／ 👨‍⚕️ 外科醫師（標記）

- 有側別的手術 → 確認外科醫師已標記（YES 字樣）
- 確認標記位置與排程一致
- 👩‍⚕️ **確認重點**：
  - 核對標記部位與手術排程表（姓名 + 術式 + 側別 + 部位）完全一致
  - ⚠️ 標記缺失 → 聯繫外科醫師補標（不可放行至開刀房）
  - ⚠️ 不可由護理師代標 → 必須由手術醫師親自標記（JCI/TNHIA 規範）
  - 標記已褪色 / 模糊 → 仍需請醫師重新標記

### [B-08] 建立靜脈通路（IV line）— 如病房端需建立
**執行者**：👩‍⚕️ 病房護理師

- 選擇穿刺部位
  - 避免手術側上肢
  - 避免 AV fistula 側
  - 避免淋巴結清除側
  - 避免已有傷口 / 感染的區域
- 選擇管針口徑
  - 一般手術：20G 或 22G
  - 預期大出血 / 輸血需求：18G（或更粗）
  - 兒科 / 困難血管：24G
- 穿刺
  - 消毒 → 止血帶 → 穿刺 → 確認回血 → 固定
  - ⚠️ 失敗 → 換部位重試（最多 2–3 次，之後上報或改 OR 內處理）
- 連接點滴（LR / NS）
- 調整流速
- 記錄管路位置 + 口徑 + 建立時間
- 👩‍⚕️ **護理師注意**：
  - 穿刺失敗 ≥ 2 次 → 換人或上報（避免反覆穿刺造成病人痛苦 + 血管耗損）
  - 記錄穿刺次數 + 最終建立位置
  - 固定牢靠：透明敷料（Tegaderm）+ 日期標籤
  - 通知交班：管針口徑 + 位置 + 是否有回血困難（讓麻醉端評估是否需重建）

### [B-09] 辨識手環確認
**執行者**：👩‍⚕️ 病房護理師

- 核對手環資訊完整（姓名 / 病歷號 / 生日 / 過敏標示）
- 有過敏 → 紅色過敏手環
- 跌倒高風險 → 黃色手環（依院規）
- ⚠️ 手環脫落或模糊 → 重新製作

### [B-10] 備血確認（如需要）
**執行者**：👩‍⚕️ 病房護理師

- 確認已完成血型(ABO/Rh) + 交叉配血
- 確認備血單位數（依手術需求）
- 確認血庫已備妥 / 或需通知血庫
- ⚠️ 未備血但手術有出血風險 → 緊急通知

## B3. 出發前

### [B-11] 病歷資料夾整理
**執行者**：👩‍⚕️ 病房護理師

- 麻醉同意書 ✓
- 手術同意書 ✓
- 自費同意書 ✓（如有）
- 麻醉術前評估表 ✓
- 護理交接紀錄 ✓
- 最新檢驗報告 ✓
- 影像報告（如適用）✓
- 備血記錄（如適用）✓
- 👩‍⚕️ **Checklist 確認**：
  - 逐項勾選，不可遺漏
  - 文件不齊 → 立即聯繫補件
  - 同意書日期過期（> 30天 依院規）→ 需重新簽署

### [B-12] 通知開刀房
**執行者**：👩‍⚕️ 病房護理師

- 電話 / 系統通知：「XX 床病人 ready」
- 確認傳送人員已排定
- 👩‍⚕️ **通知內容**：
  - 病人姓名 + 病歷號
  - 手術名稱
  - 病人狀態（需推床/可輪椅）
  - 特殊注意事項（O2 使用中 / 隔離 / 管路多）

### [B-13] 搬運前最後確認
**執行者**：👩‍⚕️ 病房護理師

- 病人姓名 vs 排程
- 手術部位標記 in place
- IV 通暢
- 禁食確認
- 所有文件已備齊
- 個人物品已處理
- 👩‍⚕️ **最後動作**：
  - 請病人上廁所（最後一次機會）
  - 確認病人情緒穩定（嚴重焦慮 → 通知麻醉科）
  - 通知家屬等候地點

### [B-14] 等待傳送人員
**執行者**：👩‍⚕️ 病房護理師 + 🚶 傳送人員

- 傳送人員抵達
- 護理師 + 傳送人員共同確認病人身份
- 交接病歷資料夾
- 👩‍⚕️ **交接要點**：
  - 口頭交班：病人姓名、手術、注意事項
  - 確認點滴架已安裝
  - 確認引流管/導尿管（如有）已固定不會被拉扯
  - 確認氧氣（如有）已備妥（攜帶型 O2 或 E-cylinder）
