# Phase C：前往開刀房 & 報到

> **步驟代碼**：C-01 ~ C-08（共 8 步）  
> **主要執行者**：🚶 傳送人員 ／ 👩‍⚕️ 前台報到人員 ／ 👩‍⚕️ 病房護理師  
> **時機**：病房出發 → 抵達開刀房前台完成報到  
> **基線**：🟢 常規刀（Elective）

---

## C1. 轉送至開刀房

### [C-01] 從病房出發
**執行者**：🚶 傳送人員 + 👩‍⚕️ 病房護理師（交接後由傳送負責）

- 推床 / 輪椅（依病人行動能力）
- 確認點滴架固定
- 確認引流管/導尿管固定（如有）
- 被單覆蓋保暖 + 維護隱私
- 👩‍⚕️ **病房護理師在出發前已完成**：
  - 確認傳送人員身份 + 核對排程
  - 交付病歷資料夾（見 B-11）
  - 口頭交班（姓名、手術、注意事項）

### [C-02] 移動路線
**執行者**：🚶 傳送人員

- 搭乘專用電梯至手術樓層
- 行經走廊到開刀房管制區入口
- ⚠️ 轉送中觀察病人狀態（意識、呼吸、IV 通暢）
- ⚠️ 安撫病人情緒（簡單告知：「我們要搭電梯去開刀房了」）

---

## C2. 開刀房前台報到

### [C-03] 抵達前台（Reception）
**執行者**：🚶 傳送人員

- 傳送人員將推床停靠前台窗口
- 告知前台人員：「XX 病人報到」

### [C-04] 前台核對身份
**執行者**：👩‍⚕️ 前台報到人員

- 掃描手環條碼 或 人工查閱
- 核對排程系統：姓名 + 病歷號 + 預定手術名稱 + 刀房號
- 口頭詢問病人：「請問您的名字？生日？」
- ⚠️ 資訊不符 → 暫停 → 釐清後才可繼續

### [C-05] 核對生理參數
**執行者**：👩‍⚕️ 前台報到人員

- 查看病房端最後一套生命徵象（BP / HR / RR / SpO2 / Temp）+ 現場再次測量
- 現場再次確認 NPO 時間
- 現場再次確認有沒有抽菸喝酒嚼檳榔
- 確認有無使用氧氣 → 檢查氧氣儲量是否需要盡快進入開刀房
- 記錄於開刀房報到表
- ⚠️ 異常值（例如 SBP > 180, HR > 120, SpO2 < 92, 發燒 > 38°C）
  - → 通知麻醉醫師評估
  - → 麻醉醫師決定：可繼續 / 需先處理 / 需延後手術

### [C-06] 核對病歷表單 — 逐一清點
**執行者**：👩‍⚕️ 前台報到人員

- 麻醉照會單 / 術前評估表 → ✓ 已填寫完整
- 麻醉同意書 → ✓ 已簽署（病人 + 醫師）
- 手術同意書 → ✓ 已簽署
- 自費同意書 → ✓ 已簽署（如有自費品項）
- 護理交接紀錄 → ✓ 已填寫
- 術前檢查報告 → ✓ 已附上
- 備血記錄 → ✓（如需要）
- ⚠️ 文件缺漏 → 聯繫病房補件 → 暫不進入刀房

### [C-07] 前台系統登錄
**執行者**：👩‍⚕️ 前台報到人員

- 系統登記病人「已報到」
- 記錄報到時間
- 更新刀房狀態看板
- 通知對應刀房團隊：「病人已到」

### [C-08] 判斷去向
**執行者**：👩‍⚕️ 前台報到人員（依系統狀態決定）

- 🅰️ **到等候區（Holding Area）** —— 若：
  - 該刀房前一台手術尚未結束
  - 麻醉 / 外科團隊尚未就位
  - 需先在等候區做進一步評估
- 🅱️ **直接進入指定刀房** —— 若：
  - 刀房已清潔 / 備妥
  - 團隊已就位
  - 是該刀房第一台
