# Phase L：術後照護 & 紀錄結案

> **步驟代碼**：L-01 ~ L-10（共 10 步，含新增 🆕 L-09、L-10）  
> **主要執行者**：👨‍⚕️ 麻醉醫師 ／ 👩‍⚕️ 麻醉護理師  
> **時機**：病人離開 PACU 後 → APS追蹤 → 術後訪視 → 紀錄結案 → 刀房清理  
> **基線**：🟢 常規刀（Elective）

---

## L1. 術後照護

### [L-01] 急性疼痛服務 (APS)
**執行者**：👨‍⚕️ 麻醉醫師（APS團隊） + 👩‍⚕️ 麻醉護理師（APS巡房協助）

- APS 團隊巡房
- 評估 PCA 使用狀況
- 調整 PCA 設定（如需要）
- Epidural 巡查
  - 阻滯高度
  - 感覺/運動功能
  - Catheter site 感染/脫位
- Multimodal 止痛調整
- ⚠️ PCA 過度使用 → 評估呼吸抑制風險
- ⚠️ 止痛不足 → rescue dose + 調整方案
- 👩‍⚕️ **麻醉護理師（APS 巡房）**：
  - 檢視 PCA 用量紀錄（bolus 次數、total dose）
  - 評估病人疼痛（NRS）+ 副作用（噁心、嗜睡、搔癢）
  - 確認 epidural catheter 位置 + 敷料完整性
  - 記錄巡查結果 → 回報醫師

### [L-02] 術後訪視
**執行者**：👨‍⚕️ 麻醉醫師（24–48 hr 內）

- 評估內容
  - 術後疼痛狀況
  - PONV 狀況
  - 麻醉相關併發症篩檢
    - 喉嚨痛 / 聲音沙啞
    - 牙齒損傷
    - 嘴唇 / 舌頭損傷
    - 周邊神經損傷（ulnar / brachial plexus / peroneal）
    - 術後認知功能變化（高齡）
    - 脊椎麻醉後頭痛（PDPH）
  - 病人滿意度
- 記錄於術後訪視紀錄

### [L-03] 併發症追蹤（如發生）
**執行者**：👨‍⚕️ 麻醉醫師

- PDPH → epidural blood patch 諮詢
- 神經損傷 → 神經科會診 + 追蹤
- 術後譫妄 → 非藥物介入 + 藥物（Haloperidol / Dexmedetomidine）
- 呼吸事件 → 加強監測 / ICU 轉入

---

## L2. 紀錄完成

### [L-04] 麻醉紀錄完成
**執行者**：👩‍⚕️ 麻醉護理師（填寫） + 👨‍⚕️ 麻醉醫師（審核 + 簽名）

- 電子 AIMS 或紙本
- 確認所有時間點/藥物/事件已記錄完整
- 確認 I/O 計算正確
- 麻醉醫師簽名
- 👩‍⚕️ **護理師**：
  - 最終核對 AIMS：所有欄位填完（誘導時間、拔管時間、轉出時間）
  - 確認 I/O balance 正確（晶體 + 膠體 + 血品 vs 出血 + 尿量）
  - 確認所有事件 free text 已記錄
  - 提交審核 → 醫師確認後電子簽章

### [L-05] 管制藥品結案
**執行者**：👩‍⚕️ 麻醉護理師（主責） + 👨‍⚕️ 麻醉醫師 or 另一位護理師（見證）

- 未使用的管制藥品歸還
- 兩人核對銷毀（殘餘 Fentanyl 等）
- 管制藥品紀錄簿簽名
- ⚠️ 數量不合 → 報告管制藥品管理人
- 👩‍⚕️ **護理師 — 管制藥品結案流程**：
  1. 清點已使用 syringe（剩餘量 + 已推量 = 領取量）
  2. 未開封安瓿 → 歸還電子管制藥品櫃
  3. 殘餘藥液 → 兩人面前丟棄（倒入尖銳物品盒或指定容器）
  4. 雙方簽名確認（電子系統 + 紙本紀錄）
  5. ⚠️ 數量不合 → 不可離開 → 立即報告 → 啟動追查
  6. 完成 → 結案 → 電子系統鎖定本次紀錄

### [L-06] 自費品項結案
**執行者**：👩‍⚕️ 麻醉護理師 + 👩‍⚕️ 流動護理師

- 確認使用品項 vs 同意書一致
- 未使用品項退還
- 費用核算

### [L-07] 不良事件通報（如有）
**執行者**：👨‍⚕️ 麻醉醫師 + 👩‍⚕️ 麻醉護理師

- 院內通報系統填報
- 困難氣道 → 記錄 + 通知病人 + 建議戴 MedicAlert
- 過敏 → 記錄 + 更新過敏清單
- 設備故障 → 通報生物醫學工程
- MH → 通報 + 轉介遺傳諮詢

### [L-08] 品質紀錄
**執行者**：👨‍⚕️ 麻醉醫師 + 👩‍⚕️ 麻醉護理師

- 術中低血壓事件
- 非計畫性 ICU 轉入
- 困難氣道紀錄
- 輸血事件
- 計入品質指標追蹤

---

## L3. 刀房清理與結案 🆕

### [L-09] 術後刀房清理與設備歸位 🆕
**執行者**：👩‍⚕️ 麻醉護理師

- **麻醉機清理**
  - 關閉揮發器（已於 J-03 歸零，此處確認）
  - CO2 absorber 顏色檢查（變色 → 標記更換）
  - 呼吸迴路更換（依院規：每位病人或每日）
  - 表面擦拭消毒（面板、把手、APL valve）
- **工作台面清理**
  - 丟棄所有單次使用品（syringe、needle、suction catheter、ETT）
  - 尖銳物品盒確認未滿（滿 → 更換）
  - 藥物安瓿/空瓶集中（非管制 → 丟棄；管制 → L-05 已處理）
- **補充消耗品（為下一台做準備）**
  - □ 麻醉車藥物補充（常規用藥：Propofol、Fentanyl、Rocuronium 等）
  - □ 耗材補充（syringe 各 size、needle、IV set、ETT、LMA）
  - □ Suction canister 更換（如已滿）
  - □ 檢查 O2 / N2O / Air 中央供氣正常（如 pipeline）或鋼瓶壓力
- **借用設備歸還**
  - 借自其他刀房的設備（如額外 pump、video laryngoscope）→ 歸還原位
  - 困難氣道車 → 歸還原停放處 + 補充消耗品

### [L-10] 結案確認 🆕
**執行者**：👩‍⚕️ 麻醉護理師

- **結案 Checklist**
  - □ AIMS 紀錄完成 + 醫師已簽（L-04）
  - □ 管制藥品結案完成（L-05）
  - □ 自費品項結案完成（L-06）
  - □ 不良事件通報完成（如有，L-07）
  - □ 品質紀錄完成（如有，L-08）
  - □ 刀房清理完成（L-09）
  - □ 麻醉車 / 耗材已補充（下一台可用）
- **狀態更新**
  - 手術排程系統更新：本台結束
  - 如有下一台 → 通知等候區可準備送下一位病人
  - 如無下一台 → 關閉麻醉機待機模式 + 整理環境

---

> 📊 **Phase L 統計**：原有 8 步 → 增加 2 步（L-09、L-10）→ 共 10 步  
> 🆕 **新增步驟**：L-09 術後刀房清理與設備歸位、L-10 結案確認
