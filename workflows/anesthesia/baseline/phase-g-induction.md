# Phase G：麻醉誘導

> **步驟代碼**：G-01 ~ G-36（共 36 步，含新增 🆕 G-35、G-36）  
> **主要執行者**：👨‍⚕️ 麻醉醫師 ／ 👩‍⚕️ 麻醉護理師（協助）  
> **時機**：WHO Sign-In → 預氧合 → 誘導 → 氣道處理 → 呼吸器設定  
> **基線**：🟢 常規刀（Elective）

---

## G1. WHO Sign-In（手術安全查核 — 第一部分）

### [G-01] Sign-In 啟動
**執行者**：👨‍⚕️ 麻醉醫師（主導） + 👩‍⚕️ 麻醉護理師 + 👩‍⚕️ 流動護理師

- 在場人員：麻醉醫師 + 護理師（至少）
- 確認以下項目（口頭逐一確認）
- 👩‍⚕️ **麻醉護理師**：
  - 準備好 WHO checklist 表單
  - 協助逐項勾選確認

### [G-02] 病人確認
**執行者**：👨‍⚕️ 麻醉醫師 + 全體（口頭確認）

- 姓名（口頭 + 手環）
- 病歷號
- 手術名稱 + 部位 + 側別

### [G-03] 同意書確認
**執行者**：👨‍⚕️ 麻醉醫師 + 👩‍⚕️ 麻醉護理師

- 麻醉同意書 → 已簽署 ✓
- 手術同意書 → 已簽署 ✓

### [G-04] 手術部位標記確認
**執行者**：全體目視

- 有側別 → 標記存在 ✓
- 無側別 → N/A

### [G-05] 麻醉安全確認
**執行者**：👨‍⚕️ 麻醉醫師（口頭宣讀） + 👩‍⚕️ 麻醉護理師（逐項確認）

- 麻醉機 + 監測設備已查檢 ✓
- 困難氣道風險？→ YES / NO
  - YES → 確認設備準備（Plan A/B/C）
- 吸入風險（Full stomach）？→ YES / NO
  - YES → RSI 準備確認
- 大量失血風險（> 500ml）？→ YES / NO
  - YES → 備血確認（已 crossmatch + 血庫備妥）
- 過敏？→ 列出過敏藥物
- 👩‍⚕️ **麻醉護理師確認**：
  - 困難氣道 → F-20 困難氣道車已確認 ✓
  - RSI → Succinylcholine 或高劑量 Rocuronium 已備妥 ✓
  - 大出血 → 血庫聯繫 / 加壓袋 / 加溫器備妥 ✓

### [G-06] 脈搏血氧儀確認運作中 ✓
**執行者**：👩‍⚕️ 麻醉護理師

- 確認 SpO2 數值正常顯示中
- 向團隊口頭宣讀：「SpO2 監測正常」

## G2. 預氧合（Pre-oxygenation）

### [G-07] 面罩密合放置
**執行者**：👨‍⚕️ 麻醉醫師

- 左手 C-E grip 固定面罩
- 確認密合（無漏氣）
- ⚠️ 有鬍鬚 → 面罩邊緣塗凡士林 / 用濕紗布填充
- ⚠️ 無牙 → 義齒暫時留置幫助密合（誘導後再取出）/ 塞紗布填充臉頰

### [G-08] 設定 O2 流量
**執行者**：👩‍⚕️ 麻醉護理師（轉開 O2）+ 👨‍⚕️ 麻醉醫師（指示）

- Fresh Gas Flow 開至 100% O2
- 流量 8–10 L/min（或 flush）

### [G-09] 執行預氧合
**執行者**：👨‍⚕️ 麻醉醫師（面罩固定） + 👩‍⚕️ 麻醉護理師（監測）

- 標準：3–5 分鐘正常呼吸 或 8 次 vital capacity breaths
- 監測 EtO2（end-tidal O2）
  - 目標：EtO2 > 90%（理想 > 0.87）
- ⚠️ 特殊族群
  - 小兒：預氧合時間較短（FRC/BW 比小，desaturation 快）
  - 肥胖：頭高位（25–30° head-up / beach chair）
  - 孕婦：FRC ↓ → 快速 desaturation → 充分預氧合更重要
  - 急診 RSI：可能時間不足 → HFNC 15 L/min 同時預氧合
- 👩‍⚕️ **麻醉護理師**：
  - 持續觀察 SpO2、EtO2 數值
  - 口頭報告：「EtO2 已達 90%」
  - 安撫病人：「慢慢呼吸就好」

### [G-10] 預氧合完成確認
**執行者**：👨‍⚕️ 麻醉醫師

- EtO2 > 90% ✓
- SpO2 100% ✓
- 通知團隊：「預氧合完成，準備 induction」

## G3. 全身麻醉誘導（GA Induction）

### [G-11] 點滴確認通暢
**執行者**：👩‍⚕️ 麻醉護理師

- 快速 flush 確認 IV 通暢
- 無外滲 / 腫脹
- 向醫師報告：「IV 通暢」

### [G-12] 給予 Fentanyl（或其他 opioid）
**執行者**：👨‍⚕️ 麻醉醫師（指令） + 👩‍⚕️ 麻醉護理師（推藥 or 醫師自推）

- 劑量：1–2 μg/kg IV
- 緩慢推注（30–60 秒）
- 觀察：呼吸頻率可能下降 → 正常反應
- 👩‍⚕️ **麻醉護理師**：
  - 推藥前口頭確認：「Fentanyl XX μg，推了」
  - 記錄給藥時間

### [G-13] 等待 Fentanyl 作用（1–2 分鐘）
**執行者**：👨‍⚕️ 麻醉醫師（持續面罩 O2） + 👩‍⚕️ 麻醉護理師（監測）

- 持續監測 SpO2 / HR / BP
- 持續面罩 O2

### [G-14] 給予 Lidocaine（optional）
**執行者**：👨‍⚕️ 麻醉醫師 or 👩‍⚕️ 麻醉護理師（推藥）

- 劑量：1–1.5 mg/kg IV
- 用途：降低 Propofol 注射痛 / 抑制插管心血管反應

### [G-15] 給予 Propofol（或其他誘導劑）
**執行者**：👨‍⚕️ 麻醉醫師

- 劑量：1.5–2.5 mg/kg IV
- 調整劑量依病人狀況：
  - 老人 / ASA III-IV：減量（1–1.5 mg/kg）
  - 肥胖：用 LBW 或 adjusted BW 計算
  - 小兒：可能需較高劑量（2.5–3.5 mg/kg）
  - 血行動力學不穩：改用 Etomidate（0.2–0.3 mg/kg）或 Ketamine（1–2 mg/kg）
- 推注速度：依 BP 反應調整
- ⚠️ 注射痛 → 先走 Lidocaine / 混合 Lidocaine / 大靜脈注射

### [G-16] 確認意識喪失
**執行者**：👨‍⚕️ 麻醉醫師

- 呼喚病人姓名 → 無反應
- 睫毛反射消失
- 下顎鬆弛

### [G-17] 開始面罩通氣
**執行者**：👨‍⚕️ 麻醉醫師（面罩） + 👩‍⚕️ 麻醉護理師（必要時協助擠袋）

- 一手 C-E grip 面罩
- 另一手擠壓呼吸袋 / 開 APV
- 評估面罩通氣品質
  - 看胸廓起伏 ✓
  - 聽呼吸音 ✓
  - 看 ETCO2 波形出現 ✓
  - 看呼吸袋順從性
- ⚠️ **面罩通氣困難**
  - 雙手固定面罩（**需護理師擠袋**）
  - 插入 oral airway
  - 插入 nasal airway
  - 調整頭位（chin lift / jaw thrust）
  - ⚠️ 完全無法通氣 → 進入困難氣道流程
- 👩‍⚕️ **麻醉護理師**：
  - 準備 oral airway 在手邊（隨時遞上）
  - 醫師雙手固定面罩時 → 護理師負責擠呼吸袋
  - 監測 SpO2 + ETCO2 → 口頭報告數值

### [G-18] 給予肌鬆劑
**執行者**：👨‍⚕️ 麻醉醫師（指令） + 👩‍⚕️ 麻醉護理師（推藥 or 遞藥）

- **Rocuronium**：0.6 mg/kg（標準）或 1.2 mg/kg（RSI）
- 或 **Succinylcholine**：1–1.5 mg/kg（RSI / 極短手術 / 需快速肌鬆）
  - ⚠️ Succinylcholine 禁忌：高血鉀風險（燒傷/脊髓損傷/MH/肌病變）
- 或 **Cisatracurium**：0.1–0.2 mg/kg（肝腎不佳選擇）
- 記錄給藥時間
- 👩‍⚕️ **護理師推藥前**：口頭確認「Rocuronium XX mg，推了」

### [G-19] 等待肌鬆作用
**執行者**：👨‍⚕️ 麻醉醫師（面罩通氣） + 👩‍⚕️ 麻醉護理師（監測 TOF + VS）

- Rocuronium：60–90 秒
- Succinylcholine：45–60 秒
- 持續面罩通氣
- TOF 監測（如已建立）→ TOF = 0 確認完全肌鬆
- 此時持續注意 BP / HR 變化 → 低血壓嚴重 → 給 Ephedrine / Phenylephrine
- 👩‍⚕️ **麻醉護理師**：
  - 口頭報告 TOF 結果：「TOF = 0，完全肌鬆」
  - 口頭報告 VS：「血壓 XX/XX，心率 XX」
  - 準備喉鏡 + ETT 在手邊等待遞上

## G4. 氣道處理 — 氣管插管

### [G-20] 停止面罩通氣
**執行者**：👨‍⚕️ 麻醉醫師

- 移開面罩

### [G-21] 喉鏡插入
**執行者**：👨‍⚕️ 麻醉醫師

- 左手持喉鏡
- 右手打開病人口腔（scissor technique）
- Blade 從右嘴角進入
- 沿舌面推進至 vallecula（Macintosh）或 epiglottis 後方（Miller）
- 向前上方提起（lifting，非撬動）
- 👩‍⚕️ **麻醉護理師**：喉鏡遞給醫師（左手接）

### [G-22] 聲門暴露
**執行者**：👨‍⚕️ 麻醉醫師 + 👩‍⚕️ 麻醉護理師（BURP 協助）

- 評估 Cormack-Lehane grade（I / II / III / IV）
- Grade III/IV → 採取輔助措施
  - **BURP maneuver**（由護理師執行）
  - 換 Video 喉鏡
  - 換 Blade size
  - Bougie 輔助
- 👩‍⚕️ **麻醉護理師在此步的關鍵角色**：
  - 醫師口頭指示「BURP」→ 護理師以右手在甲狀軟骨處施壓（Backward, Upward, Rightward Pressure）
  - 調整力道依醫師口頭引導：「再往上一點」「停！就這裡」
  - 醫師需要換 blade / 換 video → 護理師立即遞上
  - 醫師需要 bougie → 護理師立即遞上

### [G-23] 插入 ETT
**執行者**：👨‍⚕️ 麻醉醫師 + 👩‍⚕️ 麻醉護理師（遞 ETT）

- 右手持 ETT（或護理師遞上）
- 看到聲帶 → ETT 通過聲帶
- 插入深度
  - 成人女性：門齒深度 21 cm
  - 成人男性：門齒深度 23 cm
  - 兒科：(Age/2) + 12（口腔）
- 拔出 stylet（護理師協助抽出）
- Cuff 充氣
  - 聽到漏氣聲消失即可（或 cuff 壓力計 < 25 cmH2O）
  - 👩‍⚕️ **護理師**：以 10ml 空針充 cuff，依醫師指示充氣量

### [G-24] 確認管位 — 5 步驟
**執行者**：👨‍⚕️ 麻醉醫師（聽診） + 👩‍⚕️ 麻醉護理師（擠袋 + 監測）

- ① ETCO2 波形出現（最重要的確認方法）
  - 連續 5–6 個正常 CO2 波形 ✓
- ② 雙側胸廓對稱起伏 ✓
- ③ 雙側肺尖聽診呼吸音 ✓
- ④ 雙側肺底聽診呼吸音 ✓
- ⑤ 上腹部聽診無氣過水聲 ✓（排除食道插管）
- SpO2 維持 ✓
- 👩‍⚕️ **麻醉護理師**：
  - 醫師聽診時護理師擠呼吸袋（穩定頻率）
  - 口頭報告：「ETCO2 有波形，SpO2 100%」

### [G-35] 誘導期護理師協助作業 🆕
**執行者**：👩‍⚕️ 麻醉護理師

> ⚠️ **流程缺口補充**：誘導期間護理師的協助工作是高密度、高時間壓力的連續作業，原始文件未獨立描述。

- **藥物遞送流程**
  - 醫師口頭指令 → 護理師拿起正確注射器 → 口頭回讀「Propofol XX mg」→ 推藥 or 遞給醫師
  - 推完每支藥立即放回固定位置（避免混亂）
  - ⚠️ 絕對不可同時拿兩支以上未標籤注射器
- **Cricoid Pressure（RSI 情境）**
  - 醫師口頭指示「cricoid pressure」
  - 護理師以拇指 + 食指在環狀軟骨施加約 30 N（≈ 3 kg）壓力
  - 持續壓住直到醫師確認 ETT 在位 + cuff 充好
  - 醫師口頭「release cricoid」→ 才鬆手
- **雙人面罩通氣（two-person technique）**
  - 醫師雙手固定面罩（兩手 E-C grip）
  - 護理師負責擠呼吸袋（每次 500-700 ml，頻率 10-12/min）
  - 觀察胸廓起伏 + ETCO2 波形確認通氣有效
- **設備遞送順序**（典型 GA 誘導）
  1. 遞喉鏡（左手接）
  2. 遞 ETT（右手接）—— 或放在醫師右手可及處
  3. 醫師說「stylet out」→ 抽出 stylet
  4. 遞 10ml 空針充 cuff
  5. 遞膠帶/固定器固定 ETT
  6. 接上呼吸迴路
- **持續監測**
  - 全程目視 monitor：SpO2、HR、BP、ETCO2
  - SpO2 < 95% → 立即口頭告知醫師
  - BP 下降 > 20% baseline → 口頭告知 + 準備 vasopressor

### [G-36] ETT 固定與紀錄 🆕
**執行者**：👩‍⚕️ 麻醉護理師

- **固定 ETT**
  - 使用膠帶（白貼布 + 透氣膠帶雙層固定）
    - 或使用商業 ETT holder（Thomas tube holder 等）
  - 固定方法：
    - 膠帶繞 ETT 一圈 → 貼於上唇或臉頰（避免壓迫嘴角）
    - 確認固定牢靠 → 輕拉測試不會滑動
  - 記錄門齒深度（如 23 cm at teeth）
- **連接呼吸迴路**
  - 將呼吸迴路 Y-piece 接到 ETT connector
  - 確認接頭卡緊（不會脫落）
  - 固定迴路管線（避免重力拉扯 ETT）
- **記錄**
  - ETT size + 門齒深度 + 固定方式
  - 插管時間
  - Cormack-Lehane grade（由醫師口述）
  - 嘗試次數（first attempt success or not）
  - 記錄於 AIMS / 麻醉紀錄
- **口腔保護**
  - 如有使用 bite block → 放置在 ETT 旁（防咬管）
  - 確認舌頭未被壓迫夾傷
  - 確認嘴唇無被膠帶壓傷

### [G-25] 固定 ETT（原步驟，整合至 G-36）
**執行者**：👩‍⚕️ 麻醉護理師

- *(已整合至 G-36，此步驟保留代碼做交叉參照)*
- → 參見 [G-36]

### [G-26] 接上呼吸器
**執行者**：👩‍⚕️ 麻醉護理師

- 呼吸迴路 Y-piece 接到 ETT
- 切換至機械通氣模式
- 確認 ventilator 開始送氣

## G5. 呼吸器初始設定

### [G-27] 設定通氣模式
**執行者**：👨‍⚕️ 麻醉醫師（設定） + 👩‍⚕️ 麻醉護理師（協助確認參數）

- Volume Control（最常用）或 Pressure Control
- 設定參數：
  - Tidal Volume (Vt)：6–8 ml/kg（用 IBW 計算）
    - IBW 男：50 + 0.91 × (身高cm - 152.4)
    - IBW 女：45.5 + 0.91 × (身高cm - 152.4)
  - Respiratory Rate (RR)：10–14 /min（維持 ETCO2 35–45 mmHg）
  - PEEP：5 cmH2O（標準起始）
  - FiO2：調整（先 50%–100%，穩定後依 SpO2 下調）
  - I:E ratio：1:2（標準）
- ⚠️ 特殊設定
  - COPD：延長呼氣比（1:3 或 1:4）
  - 肥胖：較高 PEEP（8–10）+ 可能需較高 Vt
  - 單肺通氣：見 Phase I 特殊情境

### [G-28] 確認呼吸力學
**執行者**：👨‍⚕️ 麻醉醫師 + 👩‍⚕️ 麻醉護理師（數值報告）

- Peak Inspiratory Pressure (PIP)：< 30 cmH2O 可接受
- Plateau Pressure (Pplat)：< 30 cmH2O（肺保護）
- Compliance 計算可接受
- ⚠️ PIP 過高 → 確認管路無扭折 / 支氣管痙攣 / 單側插管

### [G-29] 開始吸入麻醉藥（如選擇 inhalation maintenance）
**執行者**：👨‍⚕️ 麻醉醫師（設定濃度） + 👩‍⚕️ 麻醉護理師（轉開揮發器）

- Sevoflurane / Desflurane / Isoflurane
- 設定濃度（通常 1–1.5 MAC，依年齡調整）
- 確認 FiO2 + N2O 比例（如使用 N2O）
- 確認揮發器輸出 + 回路濃度顯示

### [G-30] 或 TIVA 設定（如選擇全靜脈麻醉）
**執行者**：👨‍⚕️ 麻醉醫師（設定） + 👩‍⚕️ 麻醉護理師（啟動 pump）

- Propofol TCI（Target-Controlled Infusion）
  - 設定 Marsh 或 Schnider model
  - 目標濃度（Ce 3–5 μg/ml）
- Remifentanil TCI
  - 設定 Minto model
  - 目標濃度（Ce 2–5 ng/ml）
- 或手動 infusion rate 設定
- BIS 目標 40–60
- 👩‍⚕️ **麻醉護理師**：
  - 確認 pump 管路連接正確
  - 確認藥物 syringe 裝妥
  - 依醫師指令啟動 pump
  - 口頭確認：「Propofol pump 開始，Ce target X μg/ml」

## G6. 區域麻醉（如選擇 RA 而非 GA）

### [G-31] Spinal Anesthesia（脊椎麻醉）
**執行者**：👨‍⚕️ 麻醉醫師（穿刺） + 👩‍⚕️ 麻醉護理師（協助擺位 + 監測）

- 病人擺位
  - 側臥位（affected side down 或 up，依 baricity）
  - 或坐姿（背部弓起「蝦米姿勢」）
- 確認監測設備運作中
- 消毒背部
  - Chlorhexidine 或 Betadine，由中心向外擴大
- 鋪無菌巾
- 觸摸定位
  - Iliac crest（相當於 L4 棘突或 L3-4 間隙）
  - 穿刺點：L3-4 或 L4-5
- 局部麻醉皮膚（Lidocaine 1% 皮下注射）
- 插入 Spinal needle（25G Whitacre pencil-point）
  - 正中或旁正中進路
  - 感覺突破韌帶層次
  - 回抽見 CSF（清澈液體）
- 慢速注入藥物
  - Heavy Bupivacaine 0.5%（hyperbaric）10–15 mg
  - ± Fentanyl 10–25 μg intrathecal
- 拔出 needle
- 病人回復仰臥位
- 紀錄穿刺時間 + 藥物 + 劑量
- 測試阻滯高度 — 每 2–3 分鐘
  - Pin-prick test 或 cold sensation（酒精棉）
  - 目標高度取決於手術類型（例如 C-section → T4 / 下肢手術 → T10）
- 等待阻滯穩定（約 10–15 分鐘）
- 持續監測 BP（spinal → 交感阻斷 → 低血壓）
  - ⚠️ 低血壓 → Ephedrine / Phenylephrine / 加速輸液
- 👩‍⚕️ **麻醉護理師角色**：
  - 協助病人擺位（側臥或坐姿），穩定支撐病人使其不晃動
  - 安撫病人：「背部弓起來，像蝦子一樣，我會扶著你」
  - 準備消毒盤 + 無菌手套 + spinal kit
  - 遞送藥物注射器
  - 穿刺完成後協助病人翻回仰臥
  - 每 2–3 分鐘量 NIBP + 報告數值
  - 測試阻滯高度（用酒精棉擦拭，問病人「冰不冰？」）

### [G-32] Epidural Anesthesia（硬膜外麻醉）
**執行者**：👨‍⚕️ 麻醉醫師（穿刺） + 👩‍⚕️ 麻醉護理師（協助）

- 擺位（同 spinal）
- 消毒 + 鋪巾
- 局部麻醉
- Touhy needle（17G / 18G）進針
  - Loss of Resistance（LOR）technique
  - LOR to saline 或 air
  - 突破黃韌帶 → 進入硬膜外腔
- 穿入 epidural catheter
  - 留置 catheter 進入硬膜外腔 3–5 cm
  - 回抽確認無血 / 無 CSF
- **Test dose**
  - Lidocaine 1.5% + Epinephrine 1:200,000（3 ml）
  - 觀察 3 min
    - ⚠️ HR 上升 > 20% → 懷疑 IV catheter
    - ⚠️ 快速運動阻滯 → 懷疑 intrathecal catheter
- 分次給藥（loading dose）
  - Bupivacaine 0.25–0.5% 或 Ropivacaine 0.5–0.75%
  - 每次 3–5 ml，間隔 3–5 min
  - ± Fentanyl epidural（1–2 μg/ml）
- 固定 catheter（膠帶固定於背部）
- 記錄 catheter depth（皮膚處刻度）
- 測試阻滯高度
- 👩‍⚕️ **麻醉護理師角色**：
  - 協助擺位 + 穩定病人
  - 準備 epidural kit + test dose 藥物
  - LOR 注射器準備（saline or air）
  - Catheter 穿入時固定 Touhy needle（醫師指示時）
  - 協助膠帶固定 catheter 於病人背部
  - Test dose 後密切監測 HR + BP

### [G-33] Peripheral Nerve Block（周邊神經阻斷）
**執行者**：👨‍⚕️ 麻醉醫師（穿刺） + 👩‍⚕️ 麻醉護理師（協助）

- 選擇阻斷類型（依手術部位）
  - 上肢：Interscalene / Supraclavicular / Infraclavicular / Axillary
  - 下肢：Femoral / Adductor canal / Sciatic / Popliteal / Ankle
  - 軀幹：TAP / PVB / ESP / Rectus sheath
- 擺位 + 消毒 + 無菌技術
- 超音波引導
  - 線性探頭（高頻 6–15 MHz）
  - 找到目標神經 + 周圍結構
  - In-plane 或 Out-of-plane approach
- 局部皮膚麻醉
- 穿刺
  - 在超音波引導下進針
  - 確認針尖位置
  - 回抽（確認無血液）
  - 注入少量 LA → 看到擴散（hydrodissection）
  - 分次注入藥物
    - Ropivacaine 0.25–0.5% 或 Bupivacaine 0.25–0.5%
    - 總量依 block 類型 + 體重計算最大安全劑量
- ⚠️ 局麻藥物中毒（LAST）徵兆監測
  - 耳鳴 / 金屬味 / 頭暈 / 抽搐 / 心律不整
  - → Intralipid 20% rescue protocol
- 👩‍⚕️ **麻醉護理師角色**：
  - 準備超音波機 + 無菌探頭套
  - 準備 block needle + LA 藥物
  - 抽好 LA 注射器 + 標籤
  - 擠壓 LA 注射器注入（醫師持針 + 超音波探頭，雙手忙）
  - 持續監測 VS + 觀察 LAST 早期症狀
  - Intralipid 位置已確認（F-11）

## G7. MAC（Monitored Anesthesia Care）

### [G-34] MAC 設定
**執行者**：👨‍⚕️ 麻醉醫師 + 👩‍⚕️ 麻醉護理師

- 標準監測建立（同 E2）
- 鼻導管 O2 供應
  - 2–4 L/min nasal cannula
  - 或高流量面罩
- 鎮靜藥物
  - Midazolam 1–2 mg IV titrate
  - ± Fentanyl 25–50 μg IV
  - ± Propofol infusion 25–75 μg/kg/min
  - 或 Dexmedetomidine infusion 0.2–1 μg/kg/hr
- 局部麻醉（外科醫師執行 or 麻醉醫師注射）
- 持續監測呼吸（ETCO2 via nasal cannula，SpO2）
- ⚠️ 隨時準備轉換為 GA（conversion to GA）
- 👩‍⚕️ **麻醉護理師**：
  - 鼻導管裝設 + O2 流量調整
  - MAC 期間負責持續觀察呼吸型態 + SpO2
  - 鎮靜藥物 titration 依醫師指令
  - ⚠️ MAC 病人較容易發生呼吸抑制（尤其合併 opioid + benzodiazepine）
  - GA 備品隨時在旁（面罩 + ETT + 喉鏡）
