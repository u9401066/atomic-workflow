# Phase E：手術室內 — 病人安頓 & 監測建立

> **步驟代碼**：E-01 ~ E-17（共 17 步，含新增 🆕 E-16、E-17）  
> **主要執行者**：👩‍⚕️ 麻醉護理師 ／ 👨‍⚕️ 麻醉醫師 ／ 👩‍⚕️ 流動護理師  
> **時機**：麻醉護理師開台準備 → 病人進入刀房 → 監測建立完成  
> **基線**：🟢 常規刀（Elective）

---

## E0. 開台前準備（病人到場前）🆕

> ⚠️ **關鍵流程缺口補充**：在實際臨床中，麻醉護理師在病人進入刀房「之前」就已抵達 OR 進行開台準備。此流程在原始文件中完全缺失。

### [E-16] 開台前刀房環境準備 🆕
**執行者**：👩‍⚕️ 麻醉護理師

- **抵達刀房**
  - 確認刀房號碼正確（對照排程表）
  - 確認前一台手術已完成清潔消毒
  - 開燈 + 調整室溫（目標 20–23°C，依手術需求）
- **環境確認**
  - 手術台乾淨、鋪好無菌單
  - 保溫毯已鋪設於手術台上（Under-body warming pad 或預先鋪好 Bair Hugger）
  - 地面乾燥無水漬（防滑）
  - 垃圾桶 / 銳器盒位置正確
- **電源與氣源確認**
  - 牆壁氧氣面板開啟 + 確認供氣正常
  - 牆壁抽吸面板開啟 + 確認抽吸力正常
  - 電源插座足夠 + 功能正常
  - 緊急電源標示確認
- **轉送設備預備**
  - 移位滑板（transfer board）備妥
  - 推床停靠空間確認
  - 手術台高度預先調整（接近推床高度）

### [E-17] 病人到場前設備定位 🆕
**執行者**：👩‍⚕️ 麻醉護理師

- **監測設備定位**
  - 生理監視器開機 + 確認各模組正常（ECG / NIBP / SpO2 / Temp / ETCO2）
  - ECG 電極、SpO2 探頭、NIBP cuff 放置於易取位置
  - BIS sensor 備妥（如計畫使用，確認有效期）
  - NMT sensor 備妥（如計畫使用）
- **IV 設備準備**
  - 點滴架（IV pole）安裝於手術台
  - 三通閥（3-way stopcock）+ 延長管組裝
  - 點滴掛上 + 排氣完成（LR or NS，依醫囑）
  - 輸液加溫器預熱（Level 1 / Ranger，如計畫使用）
- **保溫措施**
  - Forced Air Warmer 開機 + 設定溫度（通常 38°C / High）
  - 預熱手術台上的保溫被
  - 備好額外被單（病人轉移時蓋）
- **個人檢查**
  - 手部衛生（洗手 / 酒精乾洗手）
  - 確認自己的 PPE（口罩、帽子）
  - 確認排程表上的病人資料已閱覽（過敏、特殊注意）

## E1. 病人安頓

### [E-01] 推床進入刀房
**執行者**：👩‍⚕️ 麻醉護理師 + 👩‍⚕️ 流動護理師 + 🚶 傳送人員

- 推床推至手術台旁
- 煞車固定推床
- 👩‍⚕️ **麻醉護理師首要動作**：
  - 目視確認病人（對照排程 — 正確的病人進入正確的刀房）
  - 自我介紹：「你好，我是麻醉護理師 XX，等一下我會幫你裝上監測器」
  - 簡短安撫：「我們會在旁邊照顧你」

### [E-02] 移位決策
**執行者**：👨‍⚕️ 麻醉醫師（決策）

- 🅰️ **立即移到手術台** —— 一般情況
- 🅱️ **暫留推床上** —— 若需先建立 RA（如 spinal）再移位
- 決定取決於麻醉方式 + 外科醫師偏好

### [E-03] 移到手術台（如走 🅰️）
**執行者**：👩‍⚕️ 麻醉護理師 + 👨‍⚕️ 麻醉醫師 + 👩‍⚕️ 流動護理師

- 手術台高度調整 → 與推床同高
- 手術台煞車確認鎖定
- 使用滑板（transfer board）或直接搬移
- 指導清醒病人配合：「請慢慢移過來」
- 或由團隊搬移（麻醉 + 護理 + 助手）
- 👩‍⚕️ **麻醉護理師負責**：
  - 頭端指揮（count「1, 2, 3, 移」）
  - 保護病人頭頸部
  - 確保 IV 管路不被拉扯
  - 監控其他管路（Foley / drain）
- ⚠️ 注意事項
  - 管路不被拉扯（IV / Foley / drain）
  - 防止病人跌落
  - 頸椎損傷病人：固定頸圈 + 對齊搬移

### [E-04] 手術台上固定
**執行者**：👩‍⚕️ 麻醉護理師 + 👩‍⚕️ 流動護理師

- 安全帶跨膝上方固定（不過緊）
- 確認手臂位置
  - 手臂外展板固定（angle < 90°，避免 brachial plexus 損傷）
  - 或手臂收靠身體（tucked）
- ⚠️ 有 AV fistula / 淋巴結清除側 → 不放 BP cuff、不放 IV
- 👩‍⚕️ **麻醉護理師特別注意**：
  - 頭枕位置是否舒適（影響後續面罩通氣密合度）
  - 確認 IV 側手臂可供注藥操作
  - 確認 NIBP 量測側手臂無壓迫

### [E-05] 確認病人舒適
**執行者**：👩‍⚕️ 麻醉護理師

- 詢問病人是否有不適
- 頭枕位置調整
- 被單保暖 + 隱私
- 開啟保溫毯送風（E-16 已預熱）
- 簡短告知接下來流程：「我現在會幫你貼心電圖和量血壓」

## E2. 標準監測建立

### [E-06] 貼心電圖電極（ECG leads）
**執行者**：👩‍⚕️ 麻醉護理師

- 3-lead ECG：RA（右鎖骨下）、LA（左鎖骨下）、LL（左下腹/左髖）
- 或 5-lead ECG：加 V₁（第四肋間胸骨右緣）+ RL（右下腹）
- 確認貼片位置正確
- 確認皮膚乾燥（汗多 → 擦乾 → 用酒精擦拭 → 再貼）
- 確認男性胸毛 → 必要時剃除避免貼不住
- 接上 ECG 導線（顏色/位置對應）
- 開啟 monitor 確認 ECG 波形正常
  - ⚠️ 雜訊太多 → 重新貼 / 調整位置
  - ⚠️ 波形異常 → 確認是否真的心律不整 → 通報麻醉醫師

### [E-07] 裝血壓帶（NIBP cuff）
**執行者**：👩‍⚕️ 麻醉護理師

- **選擇合適 size**
  - 目測或量測上臂周徑
  - 小號 (S)：周徑 17–22 cm
  - 中號 (M)：周徑 22–32 cm
  - 大號 (L)：周徑 32–42 cm
  - 大腿用 (Thigh)：特殊需求
- **如果 size 不合**
  - 先找本間開刀房的備品架/抽屜
  - → 沒有 → 去隔壁開刀房借
  - → 還是沒有 → 去庫房（supply room）拿
  - → 庫房也沒有 → 通知護理長調貨
- **裝設位置**
  - 一般：非手術側上臂
  - ⚠️ 禁忌同側：AV fistula / 淋巴結清除 / IV 同側
  - 替代位置：前臂 / 小腿（特殊情況記錄 + 可能有誤差）
- 對齊動脈標記（cuff 上的「ARTERY」指示線對準肱動脈）
- 接上 NIBP 管線到 monitor
- **測試一次**
  - 按 NIBP 按鈕 → 充氣 → 讀取數值
  - ⚠️ 錯誤/無法量測 → 確認 cuff 鬆緊 / size / 管線連接
  - 向麻醉醫師報告第一組 BP 數值

### [E-08] 夾 SpO2 探頭（Pulse Oximeter）
**執行者**：👩‍⚕️ 麻醉護理師

- 選擇部位
  - 首選：非手術側手指（食指 / 中指）
  - 替代：腳趾 / 耳垂 / 前額反射式（新生兒/小兒/末梢循環差）
- 指甲油/光療已移除（B-06 確認）
- ⚠️ 波形不好
  - 末梢冰冷 → 搓揉手指 / 暖手 / 換耳垂
  - 周邊循環差 → 改用前額反射式 sensor
  - 電燒干擾 → 距電燒遠的肢體
- 確認 SpO2 波形 + 數值顯示正常
- 向麻醉醫師報告 SpO2 基線值

### [E-09] 拆封自費品項
**執行者**：👩‍⚕️ 麻醉護理師 + 👩‍⚕️ 流動護理師（記錄）

- 查閱自費同意書 → 確認品項清單
- 逐一拆封
  - BIS sensor（腦電圖監測）
  - 自費保溫毯（Bair Hugger 等品牌）
  - 自費溫度探頭
  - 特殊 IV catheter / 特殊 ETT
  - 其他自費耗材
- **記錄**
  - 品項名稱
  - 批號 / 序號 / 有效期限
  - 記錄於自費品項使用紀錄表
- 核對：拆封品項 vs 同意書品項 → 一致
- ⚠️ 品項不足 / 備品有誤 → 通知護理師調貨

### [E-10] 貼 BIS sensor（如使用）
**執行者**：👩‍⚕️ 麻醉護理師

- 擦拭前額皮膚（酒精棉片去油脂）
- 依包裝指示貼合位置（前額 + 太陽穴）
- 接上 BIS module cable
- 開啟 → 確認 SQI（Signal Quality Index）> 80
- ⚠️ SQI 低 → 重新貼 / 確認接觸 / 皮膚再清潔

### [E-11] 裝保溫毯
**執行者**：👩‍⚕️ 麻醉護理師

- 選擇保溫毯類型
  - 上半身型（上肢 + 胸部覆蓋）
  - 下半身型（下肢覆蓋）
  - 全身型
  - 選擇取決於手術部位暴露範圍
- 連接保溫毯機器（Forced Air Warmer）
  - 機器已在 E-16 預熱
  - 接上保溫毯管線
  - 確認送風正常（手感暖風從毯子出來）
- ⚠️ 機器故障 → 換一台 / 從隔壁刀房借 / 通知工程

### [E-12] 裝溫度探頭（Temp probe）
**執行者**：👩‍⚕️ 麻醉護理師（放置） + 👨‍⚕️ 麻醉醫師（指示）

- **GA 病人**（誘導後放置）
  - 鼻咽溫度探頭（最常用，GA 插管後置入）
  - 食道溫度探頭（食道聽診器含溫度探頭）
  - 直腸溫度探頭（某些情境）
- **RA / MAC 病人**
  - 腋窩溫度探頭
  - 皮膚溫度貼片
  - 紅外線耳溫
- 接上 monitor → 確認讀數合理
- 👩‍⚕️ **麻醉護理師**：先準備好所需探頭，等待醫師指示放置時機

### [E-13] 裝 NMT sensor（肌鬆監測，如計畫使用）
**執行者**：👩‍⚕️ 麻醉護理師

- TOF-Watch / TOF-Cuff
- 貼電極於 ulnar nerve 路徑
- 校正基線（誘導前做 calibration）
- 或規劃誘導後才校正

## E3. 進階監測建立（如需要）

### [E-14] 建立 Arterial line（A-line）
**執行者**：👨‍⚕️ 麻醉醫師（穿刺） + 👩‍⚕️ 麻醉護理師（協助）

- 選擇部位
  - 首選：非慣用手橈動脈（radial artery）
  - 替代：尺動脈、肱動脈、股動脈、足背動脈
- Allen's test（如穿刺橈動脈）確認尺動脈灌流
- 消毒 + 鋪巾
- 局部麻醉（清醒病人）
- 穿刺
  - 直接穿刺法 或 超音波引導
  - 確認動脈血回流
  - 導入 catheter
- 連接 transducer + 沖洗管路（heparinized saline）
- Zeroing（歸零校正）—— transducer 對齊 phlebostatic axis
- 確認動脈波形正常
- 固定 + 記錄
- 👩‍⚕️ **麻醉護理師協助**：
  - 事先準備 A-line kit（catheter + transducer + 加壓袋 + heparinized NS）
  - transducer 排氣完成
  - 協助消毒鋪巾
  - 準備固定膠帶
  - 穿刺成功後協助連接管路 + zeroing
  - 記錄建立時間 + 部位

### [E-15] 建立 CVC（Central Venous Catheter）
**執行者**：👨‍⚕️ 麻醉醫師（穿刺） + 👩‍⚕️ 麻醉護理師（協助）

- 選擇部位
  - 首選：右頸內靜脈（R-IJV）
  - 替代：鎖骨下靜脈、股靜脈、左 IJV
- 超音波確認血管位置 + 大小 + 通暢
- 消毒 + 鋪巾（最大無菌屏障）
- 局部麻醉（清醒病人）
- 超音波引導穿刺（Seldinger technique）
- 導入 guide wire → 確認位置（超音波看到 wire 在血管內）
- 擴張 → 導入 catheter
- 固定 + 縫合
- 沖洗各 port
- 確認回血
- 接上 CVP transducer（如需要）+ zeroing
- ⚠️ 術後需 CXR 確認位置 + 排除氣胸
- 👩‍⚕️ **麻醉護理師協助**：
  - 事先準備 CVC kit（triple-lumen catheter / introducer）
  - 準備超音波機 + 無菌探頭套
  - 協助最大無菌屏障（帽子、口罩、無菌衣、手套、大單）
  - 遞送 guide wire、dilator、catheter
  - 沖洗各 port + 確認回血
  - 協助固定 + 縫合準備（如持針器 + 縫線）
  - 記錄建立時間 + 部位 + catheter 長度
