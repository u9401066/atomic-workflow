# Phase F：麻醉準備 — 機器 / 藥物 / 氣道設備

> **步驟代碼**：F-01 ~ F-20（共 20 步，含新增 🆕 F-18、F-19、F-20）  
> **主要執行者**：👩‍⚕️ 麻醉護理師 ／ 👨‍⚕️ 麻醉醫師（確認/監督）  
> **時機**：開台準備（病人到場前即開始）→ 誘導前完成  
> **基線**：🟢 常規刀（Elective）

> ⚠️ **重要說明**：Phase F 的大部分工作（F-01 ~ F-17）實際上與 Phase E 的開台準備（E-16、E-17）**同時進行**，都在病人到場之前完成。麻醉護理師會在開台時一併完成機器檢查、藥物準備和氣道設備準備。

## F0. 管制藥品領取 🆕

> ⚠️ **關鍵流程缺口補充**：管制藥品（如 Fentanyl、Morphine）的領取流程有嚴格的法規要求，原始文件未獨立描述此流程。

### [F-18] 管制藥品領取流程 🆕
**執行者**：👩‍⚕️ 麻醉護理師

- **查閱醫囑**
  - 確認該台手術的麻醉計畫中需要哪些管制藥品
  - 常見品項：Fentanyl、Morphine、Alfentanil、Remifentanil、Midazolam
  - 確認劑量範圍（依病人體重 / 手術類型）
- **管制藥櫃操作**
  - 至管制藥品存放區（通常為電子管制藥櫃，如 Pyxis / Omnicell）
  - 指紋 or 密碼 + 工號登入
  - 選擇病人 → 選擇藥品 → 確認品項 + 數量
  - 取出藥品
  - 系統自動記錄（取藥者 / 時間 / 品項 / 數量）
- **簽帳**
  - 管制藥品登記簿簽名（紙本 or 電子）
  - 記錄：日期 / 時間 / 病人姓名 / 藥名 / 數量 / 簽名
- **運送**
  - 藥品放入專用攜帶盒（上鎖或封口）
  - 直接攜帶至刀房（不可轉交他人、不可暫放公共區域）
- ⚠️ 管制藥品**全程不離身**直到注射或歸還
- ⚠️ 取藥後發現不需要 → 原封退回 + 退藥簽名

### [F-19] 藥物核對與標籤作業 🆕
**執行者**：👩‍⚕️ 麻醉護理師（準備） + 👨‍⚕️ 麻醉醫師（確認）

- **抽藥環境**
  - 在乾淨、光線充足的工作檯面操作
  - 手部衛生（酒精乾洗手）
  - 一次只抽一種藥（避免混淆）
- **抽藥流程**
  - 確認安瓶/vial 上的：藥名 + 濃度 + 有效期限
  - 使用正確 size 注射器
  - 消毒 vial 橡膠塞（如使用 vial）
  - 抽取正確劑量
  - ⚠️ 抽完立即丟棄空安瓶（管制藥空安瓶需保留至結案清點）
- **標籤黏貼**（ISMP / ASA 國際顏色規範）
  - 每支注射器抽完**立即**貼標籤
  - 標籤內容：藥名 + 濃度 + 日期 + 時間 + 抽藥者姓名
  - 顏色規範：
    - 🔵 藍色 = Opioids（Fentanyl, Morphine）
    - 🟡 黃色 = Induction agents（Propofol）
    - 🔴 紅色 = Muscle Relaxants（Rocuronium, Succinylcholine）
    - 🟠 橘色 = Vasopressors（Ephedrine, Phenylephrine）
    - ⚪ 白色 = Local anesthetics（Lidocaine）
  - ⚠️ **永遠不使用未標籤的注射器**
- **醫師 Double-check**
  - 所有藥物抽好後 → 向麻醉醫師口頭報告
  - 醫師目視確認標籤 + 濃度
  - 高危險藥物（如 Succinylcholine / Epinephrine）→ 雙人確認

## F1. 麻醉機檢查

### [F-01] 麻醉機開機自檢
**執行者**：👩‍⚕️ 麻醉護理師（執行） + 👨‍⚕️ 麻醉醫師（最終確認）

- 開啟電源
- 等待自檢程序完成（各品牌不同：Dräger / GE / Mindray）
- 確認自檢通過（綠燈 / OK 顯示）
- ⚠️ 自檢失敗 → 不可使用 → 通知生物醫學工程 → 換機器
- 👩‍⚕️ **麻醉護理師**：
  - 通常為第一台時執行完整自檢
  - 台間只做短版 checkout（確認迴路 + 呼吸囊 + 抽吸）
  - 記錄自檢結果於麻醉機 logbook

### [F-02] 氣體供應確認
**執行者**：👩‍⚕️ 麻醉護理師

- 中央氣體供應連接確認
  - O2 管線（綠色）接頭密合
  - N2O 管線（藍色）接頭密合
  - Air 管線（黃色）接頭密合
- 備用 O2 鋼瓶
  - 確認有裝
  - 確認壓力足夠（> 1000 psi / > 70 bar）
  - ⚠️ 低壓 → 更換鋼瓶
- 👩‍⚕️ **護理師注意**：若鋼瓶壓力偏低（< 500 psi），即使可用也應更換或通知補充

### [F-03] 呼吸迴路連接 & 漏氣測試
**執行者**：👩‍⚕️ 麻醉護理師

- 組裝呼吸迴路（吸氣管 + 呼氣管 + Y-piece）
- 接上 CO2 absorbent canister
  - 確認鈉石灰顏色（未變色 = 堪用）
  - ⚠️ 已變色（紫色/灰色）→ 更換
- 封閉 Y-piece → 執行 leak test
  - 系統自動加壓 → 確認漏氣率在允許範圍
- 接上 HME filter（Heat Moisture Exchanger）
- 確認 APL valve 功能正常

### [F-04] 揮發器（Vaporizer）確認
**執行者**：👩‍⚕️ 麻醉護理師

- 確認揮發器安裝正確
- 檢查液面（Sevoflurane / Desflurane / Isoflurane）
  - ⚠️ 液面低 → 補充（護理師可執行或通知技術人員）
- 確認揮發器 lock 正確
- Desflurane 加熱型揮發器：確認已加熱完成（需約 10 min 預熱）

### [F-05] 抽吸系統確認
**執行者**：👩‍⚕️ 麻醉護理師

- 牆壁抽吸連接（或可攜式抽吸機）
- 接上抽吸管（Yankauer）
- 測試抽吸力（手封住管口 → 壓力顯示正常 / 聽到抽吸音）
- 確認收集瓶安裝正確
- 👩‍⚕️ **護理師**：抽吸管放置於麻醉醫師右手側，隨時可取

### [F-06] 備用甦醒球（Self-inflating bag / Ambu bag）
**執行者**：👩‍⚕️ 麻醉護理師

- 確認在手邊可及處（通常掛在麻醉機側面）
- 確認功能良好（擠壓 → 感覺到 valve 出氣）
- 接上 O2 管線（可接中央或鋼瓶）
- 👩‍⚕️ **護理師**：每台手術都要確認 Ambu bag 功能正常，這是斷電 / 麻醉機故障時的生命線

## F2. 藥物準備

### [F-07] 抽取誘導藥物
**執行者**：👩‍⚕️ 麻醉護理師（抽藥） + 👨‍⚕️ 麻醉醫師（確認）

- **Propofol**
  - 20ml 注射器
  - 記錄濃度（10 mg/ml）
  - 標籤：藥名 + 濃度 + 日期 + 時間 + 抽藥者姓名
- **Fentanyl**（管制藥品，F-18 已領取）
  - 注射器（通常 3–5 ml）
  - 記錄濃度（50 μg/ml）
  - 標籤
- **Rocuronium**（或 Cisatracurium / Succinylcholine）
  - 注射器
  - 記錄濃度
  - 標籤
- **Lidocaine**（optional，降低 Propofol 注射痛 / 降低插管反應）
  - 注射器
  - 記錄濃度（20 mg/ml）
  - 標籤
- ⚠️ 所有注射器必須貼標籤（顏色依 F-19 規範）
- ⚠️ 管制藥物空安瓶保留（結案時清點用）

### [F-08] 準備急救藥物
**執行者**：👩‍⚕️ 麻醉護理師

- **Atropine** 0.5 mg/ml — 注射器抽好或安瓶備妥
- **Ephedrine** — 稀釋至 5 mg/ml（通常 1ml Ephedrine 50mg + 9ml NS）
- **Phenylephrine** — 稀釋至 100 μg/ml（通常 1ml PE 10mg + 99ml NS 配成 bag）
- **Epinephrine** — 1:10,000 稀釋（1mg/10ml）安瓶或預充式備
- ⚠️ 急救藥物放置位置統一 + 隨手可取
- 👩‍⚕️ **護理師排列習慣**：
  - 急救藥排在工作檯最靠手邊的位置
  - 慣用排列順序一致（如左到右：Atropine → Ephedrine → PE → Epi）
  - 告知醫師藥物放置位置

### [F-09] 準備持續輸注藥物（如計畫使用）
**執行者**：👩‍⚕️ 麻醉護理師（配製） + 👨‍⚕️ 麻醉醫師（設定參數）

- Remifentanil infusion（稀釋 + 輸注幫浦設定）
- Propofol infusion（TIVA 情境）
- Dexmedetomidine infusion
- Ketamine infusion（低劑量止痛）
- Insulin infusion（DM 病人）
- ⚠️ 所有 infusion line 標籤 + 輸注幫浦程式確認
- 👩‍⚕️ **護理師**：
  - 配製完成後向醫師報告濃度
  - Infusion pump 裝好管路但不啟動（等醫師指令）
  - 確認 pump 已 prime（管路排氣完成）

### [F-10] 準備止吐藥（如高 PONV 風險）
**執行者**：👩‍⚕️ 麻醉護理師

- Ondansetron / Granisetron
- Dexamethasone（誘導時給）
- Droperidol（低劑量）
- 備妥但不抽（或抽好標籤待用）

### [F-11] 準備特殊藥物（如需要）
**執行者**：👩‍⚕️ 麻醉護理師（確認位置） + 👨‍⚕️ 麻醉醫師（決策）

- Dantrolene 位置確認（MH cart 位置）
- Sugammadex（肌鬆逆轉）
- Intralipid 20%（LAST 急救）
- Nitroglycerin / Esmolol / Labetalol（心血管手術）
- TXA（Tranexamic acid，大出血風險）
- 👩‍⚕️ **護理師**：
  - 特殊藥物不一定事先抽好，但要確認「在哪裡、多快能拿到」
  - MH cart 位置 → 每間 OR 護理師都要知道
  - Intralipid 位置 → 尤其做 nerve block 時要確認

### [F-12] 輸液準備
**執行者**：👩‍⚕️ 麻醉護理師

- 選擇晶體液（LR / NS / Plasmalyte）
- 掛上點滴 + 排氣（確認無氣泡）
- 接上 IV extension + 3-way stopcock（三通閥）
- 輸液加溫器設定（Level 1 / Ranger 等，如預期大量輸液/輸血）
- 加壓袋準備（如需要快速輸液）
- 👩‍⚕️ **護理師**：已在 E-17 完成初步準備，此處為最終確認 + 連接病人端

## F3. 氣道設備準備

### [F-13] 喉鏡準備
**執行者**：👩‍⚕️ 麻醉護理師

- 直視喉鏡
  - 選擇 blade（Macintosh #3 或 #4 / Miller #2）
  - 確認燈泡亮（或 LED 光源正常）
  - 準備備用 blade（不同 size）
- Video 喉鏡（如 GlideScope / McGrath / C-MAC）
  - 開機 → 確認螢幕顯示
  - 選擇 blade size
  - 備用直視喉鏡在旁
- 👩‍⚕️ **護理師**：
  - 依病人評估選擇（看術前評估 Mallampati / 困難氣道標記）
  - 困難氣道病人 → 優先備好 video 喉鏡
  - 所有 blade 用完需送消毒（記錄使用）

### [F-14] 氣管內管（ETT）準備
**執行者**：👩‍⚕️ 麻醉護理師

- 選擇 size
  - 成人女性：6.5–7.0 mm ID
  - 成人男性：7.0–8.0 mm ID
  - 兒科：(Age/4) + 4（uncuffed）或 (Age/4) + 3.5（cuffed）
- 備用小一號 ETT
- Stylet 潤滑 → 插入 ETT → 彎曲成 hockey stick 形狀
- Cuff 充氣測試 → 確認無漏 → 放氣
- 潤滑 ETT 前端（水溶性潤滑劑）
- 👩‍⚕️ **護理師**：
  - 確認 ETT 包裝完整 + 有效期
  - Stylet 彎度依醫師偏好調整
  - 準備 10ml 空針（充 cuff 用）
  - 準備固定膠帶（或 ETT holder）

### [F-15] 備用氣道設備
**執行者**：👩‍⚕️ 麻醉護理師

- Oral airway（Guedel）— 選擇 size
- Nasal airway — 選擇 size（如需要）
- LMA（Laryngeal Mask Airway）/ i-gel — 選 size + cuff 測試
- Bougie（Introducer）— 備在手邊
- 困難氣道車位置確認（如高風險病人）
  - Fiberoptic bronchoscope
  - Video stylet
  - Retrograde intubation kit
  - 外科氣道工具（Cricothyrotomy kit）

### [F-20] 困難氣道車確認 🆕
**執行者**：👩‍⚕️ 麻醉護理師

- **確認困難氣道車位置**
  - 每間 OR 附近都有困難氣道車（或確認在哪個走廊/房間）
  - 確認車輪未鎖 → 可快速推入刀房
- **確認車內設備齊全**（定期盤點，每台手術前目視確認）
  - ☐ Fiberoptic bronchoscope（FOB）+ 光源 + 電池充電
  - ☐ Video stylet（如 Bonfils / Shikani）
  - ☐ 不同 size 的 LMA（#3, #4, #5）
  - ☐ Bougie（各 size）
  - ☐ 外科氣道 kit（Cricothyrotomy / Tracheostomy）
  - ☐ 經鼻清醒插管設備（lubricant / topical anesthesia / atomizer）
  - ☐ Retrograde intubation kit
- **特別注意**：
  - 術前評估為困難氣道（Mallampati III–IV / BMI > 40 / 頸椎固定 / 口腔腫瘤）→ **事先將車推至刀房內**
  - ⚠️ FOB 電池沒電 → 立即換電池或通知工程
  - ⚠️ 車內設備用過未補 → 通知護理長補貨

### [F-16] 抽吸管接上
**執行者**：👩‍⚕️ 麻醉護理師

- Yankauer suction tip
- 接上抽吸管線
- 測試抽吸功能（已在 F-05 確認牆壁端）
- 放置於麻醉醫師右手可及處

### [F-17] 面罩（Face Mask）準備
**執行者**：👩‍⚕️ 麻醉護理師

- 選擇合適 size（蓋住口鼻無漏氣）
- 成人：size 4 或 5
- 小兒：size 1–3
- 連接呼吸迴路 Y-piece
- 👩‍⚕️ **護理師**：
  - 依病人臉型大小選擇
  - 準備兩個 size 備用（避免密合不佳需更換時手忙腳亂）

## F4. 區域麻醉設備準備（如計畫 RA）

### [F-21] 區域麻醉設備準備 🆕
**執行者**：👩‍⚕️ 麻醉護理師（準備） + 👨‍⚕️ 麻醉醫師（確認計畫）

> ⚠️ 依術前評估（Phase A）確認的麻醉計畫，若包含 Spinal / Epidural / PNB，需在誘導前完成設備準備。

#### Spinal Anesthesia 設備（→ 用於 G-31）
- ☐ Spinal tray（或組合包：消毒盤 + 洞巾 + 紗布）
- ☐ Spinal needle：25G 或 27G Whitacre（pencil-point）
  - 備用：不同 size（22G Quincke 粗針備用）
- ☐ Introducer needle（18G 或 19G）
- ☐ 無菌手套（醫師 size）
- ☐ 消毒液（Chlorhexidine in alcohol 或 Betadine）
- ☐ 局部麻醉：Lidocaine 1% + 25G 皮膚穿刺針 + 注射器（皮膚浸潤用）
- ☐ 脊髓藥物：Heavy Bupivacaine 0.5%（hyperbaric）安瓶
  - ± Fentanyl 安瓶（intrathecal adjuvant）
- ☐ 1ml 或 3ml 注射器（抽取脊髓藥物用）

#### Epidural Anesthesia 設備（→ 用於 G-32）
- ☐ Epidural kit（含 Tuohy needle 17G/18G + epidural catheter + filter + LOR syringe）
- ☐ 無菌手套 + 消毒液 + 洞巾
- ☐ Loss-of-Resistance 注射器（LOR to saline 或 air — 依醫師偏好確認）
- ☐ Test dose 藥物：Lidocaine 1.5% + Epinephrine 1:200,000（3ml）
- ☐ Loading dose 藥物：Bupivacaine 0.25–0.5% 或 Ropivacaine 0.5–0.75%
  - ± Fentanyl（epidural concentration 1–2 μg/ml）
- ☐ 膠帶（Tegaderm + 固定膠帶，用於 catheter 背部固定）
- ☐ 局部麻醉（皮膚浸潤用，同 Spinal）

#### Peripheral Nerve Block 設備（→ 用於 G-33）
- ☐ 超音波機推入刀房 + 開機 → 確認螢幕正常
  - 線性探頭（高頻 6–15 MHz）→ 表淺 block
  - 弧形探頭（低頻 2–5 MHz）→ 深層 block（如 sciatic）
- ☐ 無菌探頭套 + 無菌凝膠
- ☐ Echogenic block needle（選擇長度依 block 深度）
  - 50mm（表淺 block）/ 100mm（深層 block）
- ☐ Nerve stimulator（如需電刺激輔助確認）+ 刺激針
- ☐ LA 藥物：Ropivacaine 0.25–0.5% 或 Bupivacaine 0.25–0.5%
  - 計算最大安全劑量（Ropivacaine 3 mg/kg, Bupivacaine 2.5 mg/kg）
  - 抽入 20ml 注射器 + 標籤
- ☐ 無菌手套 + 消毒液 + 洞巾
- ☐ 局部麻醉（皮膚浸潤用）
- ☐ Intralipid 20% 位置已確認（→ F-11 已確認，此處再次口頭提醒）

#### 👩‍⚕️ **護理師準備原則**：
- 依術前評估表的「麻醉計畫」決定準備哪些項目 — 不需每台都備 RA 設備
- 不確定時主動詢問醫師：「今天這台有要做 block 嗎？需要準備什麼？」
- 所有 RA 設備擺在獨立的無菌盤上 → 不與氣道設備混用
- 超音波機：確認電池充足 / 插電使用，螢幕無故障
- ⚠️ 開台前確認 RA 消耗品庫存（spinal needle / epidural kit / block needle）→ 不足時立即補貨
