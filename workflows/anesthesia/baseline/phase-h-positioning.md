# Phase H：手術擺位 & 手術開始

> **步驟代碼**：H-01 ~ H-07（共 7 步）  
> **主要執行者**：👨‍⚕️ 麻醉醫師 ／ 👩‍⚕️ 麻醉護理師 ／ 👩‍⚕️ 流動護理師  
> **時機**：誘導完成 → 擺位 → Time-Out → Incision  
> **基線**：🟢 常規刀（Elective）

---

## H1. 手術擺位

### [H-01] 誘導後擺位（如需要改變位置）
**執行者**：👨‍⚕️ 麻醉醫師 + 👩‍⚕️ 麻醉護理師 + 👩‍⚕️ 流動護理師 + 外科團隊

- 仰臥位（Supine）— 大部分手術預設
- 截石位（Lithotomy）— 婦科/泌尿科
  - 腿架安裝 + 調整
  - ⚠️ 腓總神經壓迫風險（腿架 vs 膝部墊）
  - ⚠️ 下肢區室症候群風險（手術時間 > 4 hr）
- 側臥位（Lateral decubitus）— 胸腔/腎臟手術
  - 腋下墊卷（保護腋窩血管神經）
  - 耳朵保護墊
  - 下側手臂保護
- 俯臥位（Prone）— 脊椎手術
  - ⚠️ 拔管後翻身 or 帶管翻身 → 依計畫
  - Jackson table / Wilson frame 安裝
  - 胸墊 / 骨盆墊位置
  - 頭部固定（Mayfield / 面朝下 + 額墊）
  - ⚠️ 眼睛保護（無壓迫 → 失明風險）
  - ⚠️ 管路不被壓迫（ETT / IV / A-line / CVC）
  - ⚠️ 腹部懸空（避免腹壓升高 → 出血增加）
- 坐姿（Sitting / Beach chair）— 肩關節/後顱窩
  - ⚠️ 空氣栓塞風險 → precordial Doppler / TEE
  - ⚠️ 腦灌注壓力降低 → BP 修正
- Trendelenburg / Reverse Trendelenburg
- 頭轉一側（ENT / Carotid 手術）
- 👩‍⚕️ **麻醉護理師角色**：
  - **頭端負責人** — 擺位時護理師站在頭端保護 ETT + 頭頸
  - 翻身喊口令：「1, 2, 3, 翻！」
  - 翻身過程中用手固定 ETT 不移位
  - 確認所有管路（IV / A-line / CVC / Foley）不被拉扯或壓迫
  - 俯臥位翻身後立即確認 ETT 仍在位（聽診 + ETCO2）

### [H-02] 擺位後確認
**執行者**：👨‍⚕️ 麻醉醫師 + 👩‍⚕️ 麻醉護理師 + 👩‍⚕️ 流動護理師

- 所有壓力點有墊保護
- 手臂外展 < 90°（避免 brachial plexus）
- 手指不在桌縫處（鉸鏈壓傷）
- 眼睛保護
  - 閉眼 + 眼貼
  - 俯臥位 → 確認眼球無壓迫
  - 塗防護眼藥膏（Lacrilube）
- 頭頸位置中立 or 依手術需要
- 生殖器 / 乳房無壓迫
- ⚠️ 特殊裝備
  - Sequential Compression Device (SCD) → DVT 預防
  - 開啟 SCD 機器
- 👩‍⚕️ **麻醉護理師**：
  - 頭端壓力點檢查：耳朵、前額、下巴（俯臥位）
  - 確認 ETT 未被壓到
  - 確認 IV / A-line 通暢
  - 給予眼藥膏 + 貼眼貼

### [H-03] 擺位後重新確認監測
**執行者**：👩‍⚕️ 麻醉護理師

- ECG 波形仍正常
- NIBP 重新量測（位置改變可能影響）
- SpO2 仍有訊號
- A-line 波形 + zeroing（如侵入性 BP）
- ETT 深度重新確認（尤其翻身後）
  - 重新聽診雙側呼吸音
  - ⚠️ 翻身後 ETT 可能位移 → 單側插管
- ETCO2 波形正常
- 口頭向醫師報告：「擺位後 monitor 全部正常，ETT 深度 XX cm」

## H2. Time-Out（WHO 第二部分）

### [H-04] Time-Out 啟動
**執行者**：👩‍⚕️ 流動護理師（主持） or 👨‍⚕️ 麻醉醫師（依院規）

- 所有團隊成員暫停手上動作
- 全程口頭確認 + 所有人參與

### [H-05] Time-Out 內容
**執行者**：全體團隊（口頭確認）

- □ 團隊成員自我介紹（姓名 + 角色）
- □ 病人姓名 + 病歷號
- □ 手術名稱 + 部位 + 側別
- □ 預期手術時間
- □ 預期失血量
- □ 抗生素已給予（或確認已開始計時）
  - ⚠️ 切刀前 60 min 內需完成
  - ⚠️ Vancomycin 需 1–2 hr 輸注 → 更早開始
- □ 病人過敏
- □ 影像是否已 display
- □ 麻醉特殊注意事項（困難氣道/血行動力學/大量輸血準備）
- □ 外科特殊注意事項
- □ 護理特殊注意事項（特殊器械/植入物）
- □ VTE 預防措施確認
- 👩‍⚕️ **麻醉護理師回應**：
  - 報告麻醉方式
  - 報告過敏
  - 報告特殊注意事項（困難氣道 / 大量輸血準備 / 特殊 infusion）
  - 報告抗生素給藥狀態

### [H-06] Time-Out 完成確認
**執行者**：全體

- 所有人員確認 → 「Proceed」
- ⚠️ 任何人有疑慮 → 暫停 → 釐清後再繼續

### [H-07] 手術開始 — Incision
**執行者**：👨‍⚕️ 外科醫師（劃刀） + 👩‍⚕️ 麻醉護理師（記錄）

- 外科醫師劃刀
- 麻醉醫師記錄 Incision time
- 啟動計時器（抗生素追加倒計時）
- 👩‍⚕️ **麻醉護理師**：
  - 記錄 incision time 於 AIMS
  - 設定抗生素追加 alarm（通常 3–4 hr 後提醒）
  - 記錄此時 VS baseline
