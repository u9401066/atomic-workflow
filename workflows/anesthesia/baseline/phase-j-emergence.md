# Phase J：手術結束 & 甦醒

> **步驟代碼**：J-01 ~ J-11（共 11 步）  
> **主要執行者**：👨‍⚕️ 麻醉醫師 ／ 👩‍⚕️ 麻醉護理師 ／ 👩‍⚕️ 流動護理師  
> **時機**：外科通知「即將關刀」→ 逆轉 → 甦醒 → 拔管 → WHO Sign-Out  
> **基線**：🟢 常規刀（Elective）

---

## J1. 手術結束準備

### [J-01] 減淺麻醉深度
**執行者**：👨‍⚕️ 麻醉醫師 + 👩‍⚕️ 麻醉護理師（pump 操作）

- 逐步降低吸入麻醉藥濃度
- 停止 Remifentanil infusion（或減速）
- 停止 Propofol infusion（TIVA 情境）
- 計畫長效止痛（如尚未給）
  - Morphine IV（如計畫 PCA）
  - Ketorolac / Acetaminophen IV
  - 確認 epidural / nerve block 仍有效
- 止吐藥給予
  - Ondansetron 4 mg IV（結束前 30 min）
- 👩‍⚕️ **麻醉護理師**：
  - Pump 減速 / 停止操作
  - 準備止痛藥、止吐藥（醫師指示後推藥）
  - AIMS 記錄停藥時間

### [J-02] 準備逆轉藥物（肌鬆逆轉）
**執行者**：👨‍⚕️ 麻醉醫師（選擇藥物） + 👩‍⚕️ 麻醉護理師（備藥 + TOF 測量）

- 選擇
  - **Sugammadex**：Rocuronium/Vecuronium 專用
    - TOF count ≥ 2：2 mg/kg
    - TOF count = 0（deep block）：4 mg/kg
    - 無法等待：16 mg/kg（immediate reversal）
  - **Neostigmine + Glycopyrrolate**：非去極化肌鬆劑通用
    - TOF count ≥ 2 且有衰減才能用
    - 0.04–0.07 mg/kg + Glycopyrrolate 0.2 mg per 1 mg Neostigmine
- 確認 TOF ≥ 0.9 才可拔管
- 👩‍⚕️ **麻醉護理師**：
  - TOF 測量並口頭報告結果
  - 依醫師決定抽取逆轉藥物（計算劑量 by 體重）
  - 推藥 + 記錄時間

---

## J2. 甦醒（Emergence）

### [J-03] 停止吸入麻醉藥
**執行者**：👨‍⚕️ 麻醉醫師 + 👩‍⚕️ 麻醉護理師

- 揮發器歸零
- 增加 Fresh Gas Flow（高流量 wash-out）
- 100% O2
- 👩‍⚕️ **護理師**：確認揮發器刻度歸零 + FGF 調高

### [J-04] 等待甦醒跡象
**執行者**：👨‍⚕️ 麻醉醫師（評估） + 👩‍⚕️ 麻醉護理師（觀察 + 安全）

- 自主呼吸恢復
  - 先出現不規則呼吸 → 逐漸規律
  - Tidal volume 足夠
  - RR 合理（10–20）
- 意識恢復徵象
  - 吞嚥反射恢復
  - 眼睛睜開
  - 皺眉 / 抓手 / 移動肢體
- 呼喚病人姓名 → 反應
- 指令遵從：「睜開眼睛」「握我的手」「抬頭 5 秒」
- ⚠️ delayed emergence 鑑別
  - 殘餘麻醉藥
  - 殘餘肌鬆（確認 TOF）
  - 低體溫
  - 代謝異常（血糖/電解質）
  - 腦部事件（罕見）
- 👩‍⚕️ **麻醉護理師**：
  - 站在病人旁邊，防躁動跌落（拉好側欄）
  - 口頭安撫甦醒中病人（「手術結束了」「不要拉管子」）
  - 持續監看 VS，口頭報告異常
  - 固定好所有管路（IV、drain、Foley、ETT）避免拉扯

### [J-05] 肌鬆逆轉給予（如尚未給）
**執行者**：👨‍⚕️ 麻醉醫師 + 👩‍⚕️ 麻醉護理師

- 確認 TOF
- 給予 Sugammadex 或 Neostigmine + Glycopyrrolate
- 等待 TOF ≥ 0.9

---

## J3. 拔管（Extubation）

### [J-06] 拔管條件確認
**執行者**：👨‍⚕️ 麻醉醫師（決策） + 👩‍⚕️ 麻醉護理師（協助確認）

- □ 自主呼吸規律且足夠
- □ Tidal volume > 5 ml/kg
- □ RR 10–25
- □ SpO2 > 95% on 100% O2
- □ ETCO2 正常
- □ TOF ≥ 0.9
- □ 病人能遵從指令（或有目的性肢體活動）
- □ 上氣道反射恢復（吞嚥 / 咳嗽）
- □ 血行動力學穩定

### [J-07] 抽痰
**執行者**：👩‍⚕️ 麻醉護理師（口腔抽吸） + 👨‍⚕️ 麻醉醫師（氣管內抽吸）

- 口腔抽吸（Yankauer）
- 氣管內抽吸（suction catheter via ETT）
- 確認分泌物清除
- 👩‍⚕️ **護理師**：
  - Yankauer 抽吸口咽分泌物
  - 準備好乾紗布 → 拔管後擦拭口鼻分泌物
  - 備妥面罩 + 氧氣連接（拔管後立即給氧）

### [J-08] 拔管執行
**執行者**：👨‍⚕️ 麻醉醫師（拔管） + 👩‍⚕️ 麻醉護理師（固定 + 給氧）

- 解開 ETT 固定
- Cuff 放氣
- 深吸氣後（或正壓吸氣時）拔出 ETT
- 立即給 O2
  - 面罩 / T-piece
- 觀察呼吸
  - 胸廓起伏 ✓
  - 呼吸音正常 ✓
  - SpO2 維持 ✓
  - 無 stridor ✓
- ⚠️ **拔管後喉痙攣（Laryngospasm）**
  - 100% O2 正壓通氣
  - Jaw thrust
  - 深化（Propofol 0.5 mg/kg）
  - ⚠️ 持續 → Succinylcholine 0.5–1 mg/kg
- ⚠️ **拔管後氣道水腫 / Stridor**
  - Racemic epinephrine nebulizer
  - Dexamethasone IV
  - 嚴重 → 重新插管
- 👩‍⚕️ **麻醉護理師**：
  - 拔管瞬間：固定病人頭部 + 接過 ETT 丟棄
  - 立即遞上面罩 + 開氧（已預先連接好）
  - 觀察呼吸 + SpO2 持續報告
  - ⚠️ 若喉痙攣 → 立即遞 Ambu bag + 備 Succinylcholine

### [J-09] 深拔管（Deep extubation）— 特殊情境
**執行者**：👨‍⚕️ 麻醉醫師

- 適應症：避免咳嗽（如眼科/神經外科/某些 ENT）
- 條件：自主呼吸已恢復但仍在麻醉深度
- 風險：喉痙攣/吸入 → 僅有經驗者執行

---

## J4. WHO Sign-Out（第三部分）

### [J-10] Sign-Out 執行
**執行者**：👨‍⚕️ 外科醫師 + 👨‍⚕️ 麻醉醫師/👩‍⚕️ 麻醉護理師 + 👩‍⚕️ 流動護理師

- □ 確認手術名稱（實際執行的）
- □ 紗布 / 器械 / 針計數正確
- □ 檢體標示正確
- □ 設備問題記錄
- □ 術後照護重點
  - 術後去向（PACU / ICU / 一般病房）
  - 關鍵術後醫囑
  - 術後注意事項
- 👩‍⚕️ **麻醉護理師**：
  - 報告：麻醉方式、管路、術後止痛計畫
  - 確認計數無誤後口頭回應「麻醉沒有問題」

### [J-11] 甦醒期後即時作業
**執行者**：👩‍⚕️ 麻醉護理師

- 整理麻醉台周邊（抽吸管 coil、circuit 斷開、藥物整理）
- AIMS 記錄：拔管時間、拔管時 VS、TOF 數值、甦醒品質
- 已用 syringe 集中放置（管制藥品 syringe 另外存放 → Phase L 結案）
- 確認轉送所需設備（portable SpO2、O2 tank、Ambu bag if needed）
