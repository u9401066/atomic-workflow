# Phase I：術中維持

> **步驟代碼**：I-01 ~ I-26（共 26 步，含新增 🆕 I-25、I-26）  
> **主要執行者**：👨‍⚕️ 麻醉醫師 ／ 👩‍⚕️ 麻醉護理師  
> **時機**：Incision 後 → 手術全程 → 外科通知「即將關刀」  
> **基線**：🟢 常規刀（Elective）

---

## I1. 持續監測 & 記錄

### [I-01] 生命徵象持續監測
**執行者**：👨‍⚕️ 麻醉醫師 + 👩‍⚕️ 麻醉護理師（共同監看）

- ECG：持續，注意 ST 變化 / arrhythmia
- BP（NIBP 或 A-line）：每 3–5 分鐘
- SpO2：持續
- ETCO2：持續（波形 + 數值）
- 體溫：每 15–30 分鐘
- BIS（如有）：持續，目標 40–60
- Urine output（如有 Foley）：每 30–60 分鐘記錄
- 👩‍⚕️ **麻醉護理師**：
  - 持續目視監視器（尤其醫師離開麻醉台短暫時段）
  - 異常值立即口頭通知醫師
  - BP alarm 設定上下限（醫師指示）

### [I-02] 麻醉紀錄填寫
**執行者**：👩‍⚕️ 麻醉護理師（主要） + 👨‍⚕️ 麻醉醫師（監督）

- 每 5 分鐘記錄 VS（電子 AIMS 自動或手動）
- 藥物給予記錄（時間 + 藥物 + 劑量 + 路徑）
- 輸液記錄（晶體/膠體/輸血）
- Event 記錄（特殊事件、體位變化、外科溝通）
- I/O 累計（input = 輸液 + 輸血；output = 出血 + 尿量）
- 👩‍⚕️ **護理師紀錄重點**：
  - AIMS 自動抓取的數值需確認準確性（有時候會抓到 artifact）
  - 手動事件需即時輸入（不要等到手術結束才補）
  - 每次給藥、輸液更換、輸血開始/結束都要即時記錄

## I2. 麻醉藥物管理

### [I-03] 吸入麻醉藥調控
**執行者**：👨‍⚕️ 麻醉醫師（決策） + 👩‍⚕️ 麻醉護理師（協助調整 + 記錄）

- 監測 MAC 值（age-adjusted MAC）
- 依手術刺激調整濃度
  - 強刺激（切皮/骨鋸）→ 增加
  - 弱刺激（縫合）→ 減少
- 併用 N2O 考量（降低 volatile MAC）
- ⚠️ end-tidal concentration vs dial setting 差異

### [I-04] 靜脈麻醉藥調控（TIVA）
**執行者**：👨‍⚕️ 麻醉醫師（設定） + 👩‍⚕️ 麻醉護理師（pump 操作 + BIS 監測）

- Propofol infusion rate 調整（依 BIS）
- Remifentanil infusion rate 調整（依 BP/HR/疼痛刺激）
- ⚠️ TIVA 中斷 → awareness 風險
- 👩‍⚕️ **麻醉護理師**：
  - 持續監看 BIS 數值（40–60 範圍）
  - BIS > 60 → 告知醫師（可能太淺）
  - BIS < 40 → 告知醫師（可能太深）
  - 確認 pump 藥物剩餘量 → 快用完時提前更換 syringe
  - ⚠️ Propofol syringe 用完未更換 → TIVA 中斷 → awareness

### [I-05] Opioid 追加
**執行者**：👨‍⚕️ 麻醉醫師（醫囑） + 👩‍⚕️ 麻醉護理師（推藥）

- 依手術刺激 + 生命徵象判斷
- Fentanyl bolus 或 infusion
- ⚠️ 手術快結束 → 減少長效 opioid（避免術後呼吸抑制）
- 👩‍⚕️ **護理師**：推藥確認 + 記錄時間劑量

### [I-06] 肌鬆劑追加
**執行者**：👨‍⚕️ 麻醉醫師（醫囑） + 👩‍⚕️ 麻醉護理師（推藥 + TOF 評估）

- 依 TOF 監測（TOF count 1–2 → 追加）
- 或臨床判斷（外科要求更好 relaxation / 出現 bucking）
- 記錄追加劑量 + 時間
- 👩‍⚕️ **護理師**：
  - TOF 測量結果口頭報告
  - 推藥 + 記錄
  - 累計使用量追蹤（影響後續逆轉劑量）

### [I-07] 血管活性藥物使用
**執行者**：👨‍⚕️ 麻醉醫師（決策） + 👩‍⚕️ 麻醉護理師（推藥 / 調整 infusion）

- 低血壓
  - Ephedrine bolus（5–10 mg）
  - Phenylephrine bolus（50–100 μg）或 infusion
  - ⚠️ 持續性 → 確認原因（出血/麻醉過深/過敏/心臟問題）
- 高血壓
  - 確認麻醉深度足夠
  - Esmolol / Labetalol / Nicardipine
- 心搏過慢
  - Atropine 0.5 mg（可重複）
  - Glycopyrrolate 0.2 mg
- 心搏過快
  - 確認疼痛/淺麻醉/低血容
  - Esmolol / Metoprolol
- 👩‍⚕️ **護理師**：
  - 急救藥物（F-08 已備妥）隨時可推
  - 口頭回讀：「Ephedrine 10 mg，推了」
  - 連續使用 vasopressor → 準備 infusion（護理師配製 + 接上 pump）

## I3. 輸液 & 輸血管理

### [I-08] 晶體液持續輸注
**執行者**：👩‍⚕️ 麻醉護理師（執行） + 👨‍⚕️ 麻醉醫師（醫囑）

- 維持速率：1–3 ml/kg/hr（基本需求）
- 補充禁食缺失
- 第三空間喪失（依手術類型）
- ⚠️ 過度輸液 → 組織水腫 / 肺水腫
- 👩‍⚕️ **護理師**：
  - 記錄每袋輸液更換時間 + 累計量
  - 快輸完時提前掛新袋（避免空袋進空氣）

### [I-09] 膠體液使用（如需要）
**執行者**：👨‍⚕️ 麻醉醫師（醫囑） + 👩‍⚕️ 麻醉護理師（掛上）

- Hydroxyethyl Starch (HES) — ⚠️ 部分指引已限制
- Albumin
- Gelatin
- 用於大量出血補充 intravascular volume

### [I-10] 輸血決策 & 執行
**執行者**：👨‍⚕️ 麻醉醫師（決策） + 👩‍⚕️ 麻醉護理師（核對 + 執行）

- Trigger：Hb < 7 g/dL（一般），< 8 g/dL（心臟病人）
- 下備血單 → 通知血庫
- 血品到達 → 核對（**兩人核對**）
  - 血型 / Rh / 病人姓名 / 病歷號 / 血品編號 / 有效期
- 開始輸血
  - 輸血速率（一般 15–30 min/unit）
  - 觀察輸血反應（前 15 分鐘尤其注意）
    - ⚠️ 發燒 / 寒顫 / 蕁麻疹 / 低血壓 / 血紅尿
    - → 停止輸血 → 通知血庫 → 留管路 → NS keep
- 其他血品
  - FFP（凝血功能異常 / massive transfusion）
  - Platelet（PLT < 50,000 且持續出血）
  - Cryoprecipitate（Fibrinogen < 100）
- Massive Transfusion Protocol (MTP) — 如觸發
  - 固定比例 PRBC : FFP : PLT
  - Cell salvage（自體血回收）
  - TXA 給予
- 👩‍⚕️ **麻醉護理師輸血流程**：
  - 血品送達 → 與另一人（醫師 or 護理師）雙人核對
  - 核對完簽名 → 開始輸血
  - 開始後 15 分鐘內密切觀察反應
  - 記錄開始時間 + 結束時間 + 輸血量
  - ⚠️ 輸血反應發生 → 立即停輸血 + 通知醫師 + 保留血袋送驗

### [I-11] 輸液加溫
**執行者**：👩‍⚕️ 麻醉護理師

- 大量輸液 / 輸血 → 必須加溫（37–42°C）
- 確認加溫器設定正確
- ⚠️ 低溫輸液 → 加重低體溫 → 凝血功能障礙

## I4. 通氣管理

### [I-12] 持續呼吸器監測
**執行者**：👨‍⚕️ 麻醉醫師 + 👩‍⚕️ 麻醉護理師

- ETCO2 維持 35–45 mmHg
- SpO2 > 95%（一般）
- Peak pressure / Compliance 變化
- ⚠️ ETCO2 突然下降 → 可能 PE / 心輸出量下降 / 管路脫離
- ⚠️ ETCO2 突然上升 → 可能 MH / CO2 吸收不良 / 通氣不足

### [I-13] 通氣參數調整
**執行者**：👨‍⚕️ 麻醉醫師

- RR 調整以維持 ETCO2 目標
- PEEP 調整（肺塌陷 → 增 PEEP / 通氣壓過高 → 減 PEEP）
- FiO2 下調（穩定後 → 0.3–0.5，避免 hyperoxia）
- Recruitment maneuver（如需要）

### [I-14] 抽痰
**執行者**：👩‍⚕️ 麻醉護理師 or 👨‍⚕️ 麻醉醫師

- 指徵：聽到 rhonchi / PIP 上升 / SpO2 下降
- 關閉麻醉迴路 → 斷開 → 抽吸管進入 ETT → 抽痰 → 重新接上
- ⚠️ 過程中 SpO2 可能下降 → 快速操作
- 👩‍⚕️ **護理師**：抽痰前先預氧（100% O2 30 秒），動作快速俐落

### [I-15] ETT 位置持續注意
**執行者**：👨‍⚕️ 麻醉醫師 + 👩‍⚕️ 麻醉護理師

- 尤其在手術中翻身 / 頭部移動後
- ⚠️ ETCO2 消失 / 單側呼吸音 → 立即確認管位

## I5. 特殊情境管理

### [I-16] 大出血處理
**執行者**：👨‍⚕️ 麻醉醫師（指揮） + 👩‍⚕️ 麻醉護理師（執行）

- 確認大口徑 IV 通暢（≥ 18G × 2）
- 加壓輸液 + 加溫
- 通知血庫啟動 MTP
- 動脈血氣（ABG）+ POCT 凝血（如有 ROTEM/TEG）
- 升壓藥物持續輸注
- 報告主治外科醫師出血量
- ⚠️ DIC → 積極補充凝血因子
- 👩‍⚕️ **麻醉護理師**：
  - 加壓袋加壓 + 快速輸液
  - 抽 ABG 送驗（A-line port）
  - 增開第二條 IV（如只有一條）
  - 聯繫血庫啟動 MTP
  - 持續計算 I/O balance

### [I-17] 低體溫預防 & 處理
**執行者**：👩‍⚕️ 麻醉護理師（執行） + 👨‍⚕️ 麻醉醫師（決策）

- 保溫毯持續運作
- 輸液加溫
- 刀房溫度維持（≥ 21°C）
- 體溫 < 36°C → 加強保溫措施
- ⚠️ 體溫 < 35°C → 凝血異常/藥物代謝↓/傷口感染↑

### [I-18] 心律不整處理
**執行者**：👨‍⚕️ 麻醉醫師

- 辨識類型（12-lead ECG 如需要）
- Sinus tachycardia → 找原因（疼痛/出血/淺麻醉）
- Sinus bradycardia → Atropine / Glycopyrrolate
- AF / AFL → Rate control or cardioversion
- VT / VF → ACLS protocol
- 術中 ST elevation → 通知心臟科 + 考量術中處理

### [I-19] 支氣管痙攣（Bronchospasm）
**執行者**：👨‍⚕️ 麻醉醫師 + 👩‍⚕️ 麻醉護理師（備藥）

- 深化麻醉（Sevoflurane 有支氣管擴張效果）
- Salbutamol MDI via ETT
- Epinephrine（嚴重）
- 確認 ETT 位置（排除 carina 刺激）

### [I-20] 過敏反應（Anaphylaxis）
**執行者**：👨‍⚕️ 麻醉醫師（指揮） + 👩‍⚕️ 麻醉護理師（急救藥推注）

- 停止可疑藥物
- Epinephrine 10–100 μg IV bolus（依嚴重度）
- 大量輸液
- 下調麻醉深度
- 通知團隊
- Diphenhydramine / Hydrocortisone / Salbutamol（二線）
- 術後：Tryptase 抽血

### [I-21] 惡性高熱（MH）
**執行者**：👨‍⚕️ 麻醉醫師（指揮） + 👩‍⚕️ 麻醉護理師（Dantrolene 配製） + 全體

- ⚠️ 徵兆：ETCO2 急速上升 + Tachycardia + Rigidity + 體溫上升
- 立即停止所有 trigger agents（volatile + Succinylcholine）
- 過度通氣 100% O2
- **Dantrolene 2.5 mg/kg IV bolus**（重複至症狀控制）
- 冷卻措施（冰 NS / 冰敷）
- ABG + 電解質（hyperkalemia！）
- MH cart 位置 → 立即取用
- 通知 MH hotline（如有）
- 👩‍⚕️ **麻醉護理師 — MH 關鍵角色**：
  - 立即取 MH cart（位置已在 F-11 確認）
  - Dantrolene 配製（每瓶 20 mg 需加 60 ml 注射用水，需用力搖晃溶解）
  - 體重 80 kg → 需 10 瓶 → 需多人協助配製
  - 冰 NS 快速輸注
  - 持續抽 ABG 監測

### [I-22] 術中抗生素追加
**執行者**：👩‍⚕️ 麻醉護理師（提醒 + 給藥） + 👨‍⚕️ 麻醉醫師（確認）

- 手術時間 > 2 個半衰期 → 追加
- 常見：Cefazolin 每 3–4 小時追加
- 大量出血（> 1500 ml）→ 追加
- 計時器 / alarm 提醒
- 👩‍⚕️ **護理師**：H-07 設定的 alarm 響起 → 口頭提醒醫師 → 確認後給藥 → 記錄

## I6. 與手術團隊溝通

### [I-23] 手術進度溝通
**執行者**：👨‍⚕️ 外科醫師 ↔ 👨‍⚕️ 麻醉醫師

- 外科通報：目前進度（主要步驟/關鍵時點）
- 外科需求對應
  - 「需要更好的 relaxation」→ 追加肌鬆
  - 「出血比較厲害」→ 加速輸液 + 備血
  - 「即將關刀」→ 準備 emergence
  - 「bone cement」→ 準備處理 bone cement implantation syndrome

### [I-24] 護理溝通
**執行者**：👩‍⚕️ 流動護理師 ↔ 👩‍⚕️ 麻醉護理師

- 紗布計數異常 → 暫停 → X-ray
- 需要追加供應（紗布/器械/藥物）
- 體位調整需求
- 👩‍⚕️ **麻醉護理師與流動護理師溝通要點**：
  - 需要追加藥物（如 Rocuronium 用完）→ 請流動護理師協助取藥
  - 需要其他設備（如備用 ETT / 抽吸管 / 額外 IV set）→ 流動護理師代取
  - 自費品項需追加 → 流動護理師聯繫家屬簽同意

## I7. 麻醉紀錄與交班 🆕

### [I-25] 麻醉紀錄即時維護 🆕
**執行者**：👩‍⚕️ 麻醉護理師

- **即時性要求**
  - 每次給藥 → 3 分鐘內記錄於 AIMS
  - 每次輸液更換 → 立即更新 I/O
  - 每次事件（體位更改、血氣結果、外科溝通）→ 立即記錄
  - ⚠️ 不可「事後回填」→ 法律上不被接受
- **AIMS 操作重點**
  - 藥物欄位：藥名 + 劑量 + 路徑 + 時間
  - 中注意 auto-capture 的 VS 是否有 artifact（如電燒干擾）→ 需手動標註或刪除
  - 特殊事件用「free text」欄位記錄（時間 + 內容）
  - I/O balance 持續更新
- **定期核對**
  - 每 30 分鐘核對一次 I/O 計算
  - 核對管制藥品使用量 vs 領取量（應隨時能交代）
  - 核對輸血記錄 vs 實際輸入量

### [I-26] 交班與休息替換 🆕
**執行者**：👩‍⚕️ 麻醉護理師（交班者 + 接班者）

- **適用情境**
  - 手術時間 > 4 小時
  - 護理師需要用餐 / 休息
  - 護理師需暫時離開（如上廁所 / 處理其他急事）
- **交班流程（Handoff Checklist）**
  - 🅰️ **病人資訊**：姓名、手術名稱、麻醉方式
  - 🅱️ **目前狀態**：VS 趨勢、麻醉深度、輸液進度
  - 🅲️ **藥物**：目前使用中的 infusion + rate、已給的 bolus 不需再追加的藥
  - 🅳️ **管路**：IV（位置 + 口徑）、A-line、CVC、epidural
  - 🅴️ **未處理事項**：待回報的血氣、即將追加的抗生素、即將用完的 syringe
  - 🅵️ **管制藥品**：已使用量 + 剩餘量（交班雙方清點確認）
- **交班完成確認**
  - 雙方口頭確認「交班完成」
  - 接班者確認已了解全部狀況
  - 記錄交班時間於 AIMS
